In this module, you learned how to:

- Create an Azure IoT Central custom app by using the IoT Central portal.
- Import a capability model for a custom device by using the IoT Central portal.
- Create a programming project to simulate a refrigerated truck by using Visual Studio Code or Visual Studio.
- Create a storage account, and export all the incoming data into that account by using the IoT Central portal.
- Create an Azure event hub for routing data through an Azure Stream Analytics job by using the IoT Central portal.
- Create an external app to view the data by using Power BI.

## Clean up
