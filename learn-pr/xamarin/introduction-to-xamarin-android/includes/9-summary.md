Xamarin.Android enables .NET developers to create fully native Android apps. Developers can take full advantage of the power, features, and performance of the Android platform. In this module, you've explored Xamarin.Android and its development tools, and you've built an Android application.

Use these resources to learn more about developing apps with Xamarin.Android:

- [Get started](https://docs.microsoft.com/xamarin/android/get-started/index) with Xamarin.Android.

- Learn [application fundamentals](https://docs.microsoft.com/xamarin/android/app-fundamentals/) such as working with files, databases, and more.

- Learn more about building [user interfaces](https://docs.microsoft.com/xamarin/android/user-interface).

- Work with Android [platform features](https://docs.microsoft.com/xamarin/android/platform/).

- Find out how to [build, deploy, and test](https://docs.microsoft.com/xamarin/android/deploy-test/) your applications.

- Learn how to build apps for [wearable Android devices](https://docs.microsoft.com/xamarin/android/wear/).
