Xamarin.iOS enables .NET developers to create native iOS apps. These apps can take full advantage of the power, features, and performance of the iOS platform. In this module, you learned about Xamarin.iOS, explored iOS terminology, and built an iOS application.

To learn more about developing apps with Xamarin.iOS, refer to the following documentation:

- [Get started with Xamarin.iOS](/xamarin/ios/get-started/index).

- Learn about [Xamarin.iOS application fundamentals](/xamarin/ios/app-fundamentals/), such as working with files, databases, and more.

- Learn how to [build user interfaces](/xamarin/ios/user-interface).

- Work with [iOS platform features](/xamarin/ios/platform/), such as ARKit and CoreML.

- Get instructions to [build, deployment, and test](/xamarin/ios/deploy-test/) your applications.

- Learn how to [build apps for watchOS](/xamarin/ios/watchos/).

- Learn how to [build apps for tvOS](/xamarin/ios/tvos/).
