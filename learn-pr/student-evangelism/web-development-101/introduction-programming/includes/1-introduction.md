This module covers the basics of programming languages. The topics included here apply to most modern programming languages today. You'll get a good understanding of what programming is, what programming languages are out there, and what tools to use.

## Learning objectives

  In this module, you will:
  - Explore the concept of programming.
  - Understand high-level and low-level programming.
  - Learn about the basic elements of a program.
  - Learn about the software and tools used by professional developers.
