# Project Specification
Let's talk through some more detailed project description for the use case in consideration - 'Build a Speech Translator App with Power Apps'. 

## Fabrikam Company
Fabrikam is a fictitious multi-national company with several employees and offices around the world. Fabrikam host annual Employee thanksgiving event that results into gathering of over 100,000 employees across over 70 countries for a 5 days celebration. 

Giving the diversity of the participants, there are noticeable language barrier which often affect collaboration, communication, and networking. Fabrikam is looking forward to address this challenge in this year's event and they need you to build the solution.

## Brief requirement
- App is accessible to only the employees of Fabrikam
- User should be able to select the spoken language and translation language
- For each conversation, the app should have control buttons to read out the translated text.
- Be creative with any additional features you think will make the App more usable and accessible to the employees of Fabrikam.
