In this exercise, you will learn how to import the Triage and handoff scenario template in your bot. The Triage and handoff scenario extends the built-in symptom checker and transitions the conversations to a live nurse.

## Import the Triage and handoff scenario

1. Sign in to the Azure Health Bot management portal.

2. Under **Select a template scenario** option, click on **Open scenarios template catalog**.

    :::image type="content" source="../media/3-open-catalog.png" alt-text="Open scenarios template catalog":::

3. In the Scenario Template Catalog, you can find all the available templates and select the template that you require. When you select a template, you will be able to see a preview and description of its behavior before you import it into your Health Bot.

    :::image type="content" source="../media/3-template-catalog.png" alt-text="Scenario Template Catalog":::

4. Select the **Triage and handoff** template and click **Import Template**.

    :::image type="content" source="../media/3-import.png" alt-text="Import Triage and Handoff template":::

5. Once you import the template, you will land at the designer screen.

    :::image type="content" source="../media/3-designer-screen.png" alt-text="Custom scenarios' designer screen":::

6. You can test the scenario using the Web Chat. Type your message and follow the flow of the scenario.

7. You can customize the flow either using the conversational, flow control, and advanced functionality elements on the top toolbar in the **Designer** tab or using code. The relevant code is in JavaScript, so if you are familiar with code, you can select the **Code** tab.

## Scenario Management

The new template scenario will appear in the **Manage** tab under **Scenarios** on the left toolbar.

:::image type="content" source="../media/3-manage.png" alt-text="Custom scenarios' management page":::

In the scenario management page, you can edit the scenario name, description, and ID, disable or delete the scenario. You can also click on the scenario name to navigate to the Designer screen.
