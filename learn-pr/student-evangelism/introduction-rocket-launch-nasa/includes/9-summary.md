In this module, you learned about the scope of data and how to use it to predict outcomes: 

- You took a look at the data science lifecycle and the algorithm cheat sheet.
- You learned about weather data and how it's used to make predictions about when to plan a rocket launch.
- You learned about machine learning fundamentals and the steps for training a machine learning model.
- You learned about overfitting a machine learning model and the possible outcome of creating inaccurate data.
