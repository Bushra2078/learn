This module is about channelizing the bot to Microsoft Teams. Besides this we will be demonstrating on how to build a basic app on Teams. So, it can be shared within organization. The same bot which was built in the previous sections will be now channelized and will be wrapped into a Microsoft Teams application.

## Learning objectives

In this module, you will:

- Learn about the channels available on Microsoft Azure Health Bot Platform
- Learn to channel the bot to Teams
- Build a basic Teams app for the Healthcare Bot 

## Prerequisite modules

- Introduction to Azure Health Bot
- Introduction and basic healthcare bot
- Enhanced HealthBot 
- Integrated Health bot
