This module is about channeling a bot to Microsoft Teams. You'll also learn how to build a basic app in Teams so you can share it within your organization. You'll channel the bot you created in previous modules and wrap it into a Microsoft Teams application.

## Learning objectives

In this module, you'll:

- Learn about the channels available on the Microsoft Azure Health Bot platform
- Learn how to channel a bot to Teams
- Build a basic Teams app for the healthcare bot 

## Prerequisite modules

- Introduction to Azure Health Bot
- Introduction and basic healthcare bot
- Enhanced HealthBot 
- Integrated Health bot
