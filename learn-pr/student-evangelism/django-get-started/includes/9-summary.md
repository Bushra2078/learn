Although there are many frameworks for the Python language, the Django framework has proven itself to be a worthy choice for developing applications. In this module, you installed the Django framework, created a new project, and deployed your first app, but your journey with Django has just begun. Continue through this learning path to find all of the beneficial features of the Django framework and decide if you'll add Django to your toolbelt for developing apps.

In this module, you learned:

- Why Django is great for rapid deployments.
- The difference between Django and Flask.
- The types of applications best suited for Django.
- How to install Django.
- How to create a Django project and app.
