In this module, you learned the basics of using Bash. Among other things, you:

- Learned how to use Azure Cloud Shell to create an Ubuntu Linux VM in Azure.
- Learned how to securely connect to the VM using SSH.
- Learned how Bash commands are structured.
- Learned key Bash commands, such as `ls`, `cat`, and `ps`.
- Learned how to use I/O operators in Bash commands to redirect input and output.
- Learned how to update the operating system on a Linux server.
- Learned how to find and terminate rogue processes.
- Learned how to use Bash to filter output in Cloud Shell.
- Learned how to delete an Azure resource group and everything it contains.

If you want to take your knowledge of Bash to the next level, here are three books you might find valuable:

- [The Bash Guide](https://guide.bash.academy/)
- [Learn Shell](https://www.learnshell.org/)
- [Learning the shell](http://linuxcommand.org/lc3_learning_the_shell.php)

Other notable books include [Learning the bash Shell, 3rd Edition](http://shop.oreilly.com/product/9780596009656.do), [The Linux Command Line](http://linuxcommand.org/tlcl.php), and [Advanced Bash-Scripting Guide](https://www.tldp.org/LDP/abs/html/).
