This unit summaries the main steps completed in this module.

In this module, you learned about the hardware and software requirements for the Altair emulator on the Azure Sphere.

You have a Windows 10 or Linux developer desktop and:

1. You installed the developer tools and Azure Sphere SDK.
1. You configured your Azure Sphere, including claiming the device into your Azure Sphere tenant and setting up the WiFi connection.
1. You cloned the Altair emulator repo to your desktop.
