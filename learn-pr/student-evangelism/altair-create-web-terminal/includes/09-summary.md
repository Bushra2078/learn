This unit summaries the main steps to complete this module.

## Summary.

In this module you learned how to 

1. Created property and form views for the Altair IoT Central application. From these forms you can set the parameters for the Altair emulator and chart data from the emulator.
1. How to create a Azure Static Web application for the Altair Web Terminal.
1. How to install and run the Altair virtual disk server.
