Now that you are familiar with the decision making capabilities of JavaScript, you can start thinking about the decisions you'll need to be making in the future code you write. 

Being able to make decisions in your code allows it to be responsive. It will also allow users to interact with your websites versus having them be a static page.

Congratulations, you're on your way to developing your first dynamic website using JavaScript!
