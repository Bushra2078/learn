This unit summaries the main steps completed in this module.

In this module you learned how to:

* Create property and form views for the Altair IoT Central application. From these forms you can set the parameters for the Altair emulator and chart data from the emulator.
* Create a Azure Static Web application for the Altair Web Terminal.
* How to install and run the Altair virtual disk server.
