In this unit, you'll learn how to create a cloud-based Azure IoT Central application for the Altair 8800 emulator. From the Altair Azure IoT Central application, you can set Altair parameters, review diagnostic information, and chart memory usage and weather data.

## Steps to follow

These are the steps you'll follow:

1. Create an Azure IoT Central application.
1. Set up a trusted relationship between your Azure Sphere tenant and Azure IoT Central.

In the next exercise, you'll create an Azure Iot Central application and link it to your Azure Sphere tenant.
