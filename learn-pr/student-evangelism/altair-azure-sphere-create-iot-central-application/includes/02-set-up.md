In this unit, you will learn how to create a cloud based Azure IoT Central application for the Altair 8800 emulator. From the Altair Azure IoT Central application you will be able to set Altair parameters, review diagnostic information, and chart memory usage and weather data.

## Steps to follow

An overview of the steps you'll follow is:

1. Create an Azure IoT Central application.
1. Set up a trusted relationship between your Azure Sphere tenant and Azure IoT Central.

In the next unit you will create an Azure IoT Central application.

In the next exercise you will create an Azure Iot Central application and link it to your Azure Sphere tenant.
