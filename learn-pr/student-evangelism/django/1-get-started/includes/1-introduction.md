Django is a popular server-side web framework. Its primary focus is around data-driven websites with commonly needed utilities prebuilt into the platform. Django is built on Python. It offers flexibility and a collection of community-built extensions.

This module introduces you to Django. To help you understand the Django framework, we'll take you through the types of applications that are best suited for deployment, installation, and creating your first program.

In this module you'll learn:

- Why Django is great for quick deployment of web applications.
- The difference between Django and Flask.
- What types of applications are best for Django.
- How to install Django.
- How to create a website in Django.

- Intermediate-level knowledge of Python, including:
