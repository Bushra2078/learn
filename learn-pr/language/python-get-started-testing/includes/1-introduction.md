Testing is a core practice in professional engineering. Having a good grasp of testing and leveraging its strengths is critical for developing robust software. However, it is easy to get overwhelmed by testing itself. There are different types of testing and several tools and libraries that try to achieve similar goals. 

This module clarifies some of the concepts behind testing, mainly how they exist in Python. From `unittest` (Python's standard library module) to the different types of testing and when to apply those techniques.
