In this module, we'll help you start to take your first steps with Go. We review the history of Go and the main features of the programming language. You'll install and configure Go on your workstation. We explore how you can play with Go without having to install anything on your workstation. You'll learn how Go organizes code projects to help prevent problems when you develop complex applications.

At the end of the module, you create your first "Hello World!" application in Go. You'll be ready to continue learning more about how to build applications by using Go.

## Learning objectives

In this module, you will:

- Install and configure Go on your local workstation.
- Install and configure Visual Studio Code and the Go extension.
- Explore the Go Playground.
- Create your first Go application.

## Prerequisites

- Familiarity with programming basics in general
- Familiarity with at least one programming language
- Familiarity with downloading and installing software from the internet
