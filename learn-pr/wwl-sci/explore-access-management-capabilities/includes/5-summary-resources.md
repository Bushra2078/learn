In this module, you have learned about Conditional Access and how it is used to protect resources. You have learned how Conditional Access policies use IF THEN statements together with signals to determine whether to grant access, require more information, or block access.

You also learned about Azure AD roles, including built-in admin roles, and custom roles. You learned about the concept of least privilege access, and how this protects resources.

Now that you've completed this module, you'll be able to:

- Describe Conditional Access and its benefits.
- Describe Azure AD roles.
