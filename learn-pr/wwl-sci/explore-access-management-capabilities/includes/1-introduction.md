One of the main purposes of Azure AD is to manage access. The security perimeter has shifted away from organizational boundaries to user, device, and service identities. In this module, you'll learn how Azure AD uses intelligent access management capabilities to protect organizational assets. This module considers how Conditional Access helps you to improve security. It also shows how to use Azure AD roles to control access to Azure AD resources in a directory.

In this module, you'll learn how to:

- Describe Conditional Access and its benefits.
- Describe Azure AD roles.
