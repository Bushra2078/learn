Azure introduces the concept of resources to enable organizations to create and use the services and tools they need to succeed. Admins need to ensure that resources are governed properly so that they are secure, and in line with the organization’s compliance requirements.

In this module, you’ll learn about the resource governance capabilities available for Azure.

By the end of this module, you should be able to:

- Describe some of the resource governance capabilities in Azure
- Describe the cloud adoption framework
