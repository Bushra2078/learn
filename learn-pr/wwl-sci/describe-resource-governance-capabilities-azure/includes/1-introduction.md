Azure introduces the concept of resources to enable organizations to create and use the services and tools they need to succeed. Admins need to ensure that resources are governed properly so that they're secure, and in line with the organization’s compliance requirements.

In this module, you’ll learn about the resource governance capabilities available for Azure.

After completing this module, you should be able to:

- Describe the capabilities of Azure resource locks.
- Describe the function of Azure Blueprints.
- Describe how Azure policy helps organizations assess compliance.
