Microsoft recently announced a holistic solution that provides extended detection and response (XDR) capabilities. This solution is called Microsoft Defender. Microsoft Defender integrates threat detection tools, reduces time to respond, and hardens defenses. Microsoft 365 Defender provides solutions for your data, client endpoints, Enterprise IoT, Identity, Apps, and Cloud Apps.

Microsoft security center is a centralized site where you can manage security across Microsoft identities, data, devices, apps, and infrastructure. In this module you will:

- Describe and explore Microsoft 365 Security Center
- Describe how to use Microsoft Secure store
- Explore security reports and dashboards
- Describe incidents and incident management capabilities
