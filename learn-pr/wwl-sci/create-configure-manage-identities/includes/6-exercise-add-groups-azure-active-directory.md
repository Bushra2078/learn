## Create a Microsoft 365 group in Azure Active Directory

1.  Browse to the [Azure Active Directory](https://portal.azure.com/) blade.
2.  In the left navigation, under **Manage**, select **Groups**.
3.  In the Groups blade, on the menu, select **New group**.
4.  Create a group using the following information:
    
    :::row:::
      :::column:::
        **Setting**
      :::column-end:::
      :::column:::
        **Value**
      :::column-end:::
    :::row-end:::
    :::row:::
      :::column:::
        Group type
      :::column-end:::
      :::column:::
        Microsoft 365
      :::column-end:::
    :::row-end:::
    :::row:::
      :::column:::
        Group name
      :::column-end:::
      :::column:::
        Northwest Sales
      :::column-end:::
    :::row-end:::
    :::row:::
      :::column:::
        Membership type
      :::column-end:::
      :::column:::
        Assigned
      :::column-end:::
    :::row-end:::
    :::row:::
      :::column:::
        Owners
      :::column-end:::
      :::column:::
        Assign your own administrator account as the group owner
      :::column-end:::
    :::row-end:::
    :::row:::
      :::column:::
        Members
      :::column-end:::
      :::column:::
        Assign a member of this group
      :::column-end:::
    :::row-end:::
    
    
    :::image type="content" source="../media/create-office-365-group-bdc1112a.png" alt-text="Screen image displaying the New Group blade with Group type, Group name, Owners, and Members highlighted.":::
    
5.  When complete, verify the group named **Northwest sales** is shown in the **All groups** list.
