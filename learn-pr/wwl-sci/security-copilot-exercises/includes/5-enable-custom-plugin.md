In this exercise, you experiment with a custom plugin.  You start by uploading the manifest file for your custom plugin. Uploading the manifest adds the plugin capability to Copilot.  Once the plugin is added you validte that is shows up as a system capability and start using it.  

The creation of the plugin manifest is outside the scope of this content, but you can view the contents of that file.  Also,
