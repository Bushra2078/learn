The Data Connector page shows the connectors that are connected.  The count of Windows and Linux hosts connected with an agent is available in the Log Analytics workspace.  To see your connected hosts do the following steps:

1. Select **Settings**

1. Workspace Settings (this selection will transfer you to Log Analytics)

1. In Log Analytics Settings area, select **Agents Management**

1. There are two tabs to view  - one for Windows another for Linux.
