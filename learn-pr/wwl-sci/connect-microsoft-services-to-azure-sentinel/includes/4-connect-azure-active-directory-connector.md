Gain insights into Azure Active Directory by connecting Audit and Sign in logs to Microsoft Sentinel to gather insights around Azure Active Directory scenarios. You can learn about app usage, Conditional Access policies, and legacy auth relate details using our Sign-in logs. You can get information on your Self-Service Password Reset (SSPR) usage, Azure Active Directory Management activities like user, group, role, and app management in the Audit logs table.

To view the connector page do the following steps:

1. Select Data connectors page.

1. Select **Azure Active Directory**

1. Then select the Open connector page on the preview pane.

1. Mark the checkboxes next to the logs you want to stream into Microsoft Sentinel, and select **Connect**.
