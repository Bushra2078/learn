Microsoft Defender for Endpoint (formerly named Microsoft Defender Advanced Threat Protection) is a security platform designed to prevent, detect, investigate, and respond to advanced threats. The platform creates alerts when suspicious security events are seen in an organization. Fetch alerts generated in Microsoft Defender ATP so that you can effectively analyze security events.

![Microsoft Defender for Endpoint connector page.](../media/mdeconnector.png)

To view the connector page:

- Select Data connectors page.

- Select Microsoft Defender for Endpoint (may still be called Microsoft Defender Advanced Threat Protection).

- Then select the Open connector page on the preview pane.

- Select Connect to start streaming the alerts.

- Select whether alerts from Microsoft Defender for Endpoint automatically generate incidents by selecting Enable.
