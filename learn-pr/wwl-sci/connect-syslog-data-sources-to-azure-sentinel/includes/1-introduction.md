You need to understand the Syslog connector's configuration options if you plan to deploy to your environment.  You need to know how to parse Syslog data.

In this module, you will:

- Describe the Syslog Connector deployment options.

- Run the deployment script.

- Configure the Log Analytics agent.

- Create a Parse.

## Prerequisites

- Basic knowledge of operational concepts such as monitoring, logging, and alerting

- Basic knowledge of Linux
