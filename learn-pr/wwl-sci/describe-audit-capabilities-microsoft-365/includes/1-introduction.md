There may be a time where an organization needs to investigate the actions of certain employees, perhaps due to suspicious activity. Organizations may need to track such things as who has viewed, altered, or deleted files. Audit logs can monitor this type of activity.

In this module, you will be introduced to both core and advanced auditing capabilities in Microsoft 365.
At the end of this module, you should be able to:

- Describe the core audit capabilities of Microsoft 365.
- Describe the purpose and value of Advanced Auditing.
