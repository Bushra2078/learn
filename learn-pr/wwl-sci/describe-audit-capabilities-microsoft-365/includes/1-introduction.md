There may be a time where an organization needs to investigate the actions of certain employees, perhaps because of suspicious activity. For example, organizations may need to track who has viewed, altered, or deleted files. Audit logs can monitor this type of activity.

In this module, you'll be introduced to core and advanced auditing capabilities in Microsoft 365.
After completing this module, you should be able to:

- Describe the core audit capabilities of Microsoft 365.
- Describe the purpose and value of Advanced Auditing.
