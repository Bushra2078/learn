## Scenario

Contoso, Ltd. is a financial services company in Seattle with major offices located throughout the world. Most of its compute environment runs on-premises on Windows Server. This includes virtualized workloads on Windows Server 2016 hosts.

Contoso IT staff are migrating Contoso on-premises servers to Windows Server 2019. As part of the migration, you need to determine an appropriate Business Continuity and Disaster Recovery (BCDR) solution, so that operations could continue and data could be recovered should a natural disaster occur, such as an earthquake, flooding, or fire.
