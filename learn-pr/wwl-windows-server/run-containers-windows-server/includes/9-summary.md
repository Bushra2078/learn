Using the knowledge acquired in this module, you have determined the available options for running container workloads on Windows server and how you can utilize and manage them. You have learned about Windows Server and Hyper-V containers, and the associated isolation modes, in addition to learning how to prepare the Windows Server host to run containerized workloads. You learned about Docker and how Docker interacts with Windows Server. You also learned how to manage containers by using WAC.
