1. In the Azure portal, select **Create a resource**.
2. Select **AI + Cognitive Services**.
3. Select **Computer Vision API**.
4. In the *Name* field, enter *computerVision*.
5. Select your subscription.
6. For **Location**, select the location that you've been using throughout this unit.
7. For **Pricing tier**, select *F0*.
8. Select your existing resource group.
9. Select **Create**.

That's it! You're now running an instance of the Computer Vision API, which will generate a caption and classification to be stored with the image metadata.
