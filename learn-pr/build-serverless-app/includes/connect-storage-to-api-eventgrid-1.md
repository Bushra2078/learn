Azure Event Grid connects data sources and event handlers. In this tutorial, you use Event Grid to instantly trigger your Azure functions to run image analysis each time a new photo is added to your storage container (when it was uploaded from the website).

In the tutorial, you will:
- Learn about Azure Event Grid.
- Set up Event Grid to process images that are uploaded to your website.

At the end of the tutorial, you can check your knowledge about this information with a quiz challenge.
