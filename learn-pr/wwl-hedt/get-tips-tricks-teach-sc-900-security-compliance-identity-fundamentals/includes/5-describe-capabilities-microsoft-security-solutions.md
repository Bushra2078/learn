As you are preparing to teach this module, get familiar with what the students will learn during the module. In this module, students will learn about security capabilities at Microsoft. Topics covered will include network and platform capabilities of Azure, Azure security management, and Sentinel. You'll learn about threat protection with Microsoft 365 Defender, Microsoft 365 security management, and endpoint security. 

This module consists of six lessons: 

- **Lesson 1** – Describe the basic security capabilities in Azure 

- **Lesson 2** – Describe the security management capabilities of Azure 

- **Lesson 3** – Describe the security capabilities of Azure Sentinel 

- **Lesson 4** – Describe the threat protection capabilities of Microsoft 365 Defender 

- **Lesson 5** – Describe the security management capabilities of Microsoft 365 

- **Lesson 6** – Describe endpoint security with Microsoft Intune

### Module 3 tips and tricks 

- The focus is on Microsoft 365 and Azure so consider incorporating some modules from MS-900: Microsoft 365 Fundamentals or AZ-900: Microsoft Azure Fundamentals 

- Run through labs prior to completing them in front of students 

- Explain how the services relate to the Defense in Depth model 

- Azure Security Secure Score provides recommendations and not obligations 

- Try not to get too deep into the technical aspects of each solution. This module can be VERY long if that happens 

- Explain the differences between the security roles used in services
