### Module 5 overview 

As you begin to teach this module get, familiar with what the students will learn during the module. Microsoft Dynamics 365 Project Operations empowers business users to adapt to rapidly changing business environments. It provides businesses with everything they need to manage the life cycle of a project - from project sales to invoicing and accounting. In this module, students will explore the project lifecycle within Project Operations for stocked/production order scenarios deployment type. 

This module consists of two lessons: 

- **Lesson 1** – Identify project lifecycle management capabilities in Dynamics 365 Project Operations 

- **Lesson 2** – Explore project planning and execution capabilities

### Module 5 tips and tricks 

- Review the capabilities of Dynamics 365 when it comes to supporting different project-based industries 

 - Discuss the differences between products, services, and projects 

- Discuss the different project lifecycle stages 

 - Consider utilizing Microsoft Whiteboard to illustrate the four phases of the project lifecycle 

- If time allows, consider creating a project-based organization to emphasize individual tasks in Dynamics 365 

- Develop an optimized budget and forecast – tracking costs and utilization 

 - Keep the discussion high level
