Learn about the basics of PowerShell scripting. Scripting is about automation, it's about storing steps that you do often in a file. By running files containing your scripts, you save time and reuse your existing solution. This time can now be spent that time on something more valuable. 

You'll learn some valuable constructs of the PowerShell language and how you can use them to create scripts and run said scripts.

## Learning objectives

After completing this module, you'll be able to:

- Understand how to author and run scripts.
- Use variables and parameters to make your scripts flexible.
- Apply flow control logic to make intelligent decisions.  
- Add robustness to your scripts by adding error management.
