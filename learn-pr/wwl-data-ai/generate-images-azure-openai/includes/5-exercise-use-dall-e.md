Now it's your chance to use DALL-E to generate images. In this exercise, you'll provision an Azure OpenAI service resource and use it to explore DALL-E image generation in the Azure openAI Studio playground. Then you'll use the REST API to consume a DALL-E model from a custom application.

> [!NOTE]
> To complete this exercise, you will need an [Azure subscription](https://azure.microsoft.com/free?azure-portal=true). You will also need to request (and be granted) [Azure OpenAI access](/legal/cognitive-services/openai/limited-access?azure-portal=true), including access to DALL-E.

Launch the exercise and follow the instructions.
