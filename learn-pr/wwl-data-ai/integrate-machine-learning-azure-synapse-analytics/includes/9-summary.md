You have just learned how to integrate Azure Synapse Analytics with Azure Data and AI services.

In this module, you have:

* learned how to connect an Azure Synapse Analytics workspace to an Azure Machine Learning workspace using a Linked Service.
* learned to trigger an Automated ML experiment that uses data from a Spark table
* learned how to use trained models from Azure Machine Learning for data enrichment with T-SQL Predict.
* learned how to use Azure Cognitive Services to enrich data in a SQL pool table.
