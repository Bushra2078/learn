In this module, you have learned how to use the QnA Maker service to create a knowledge base of question and answer pairs that can support an application or bot.

Now that you've completed this module, you can:

- Describe the QnA Maker service.
- Describe the differences between QnA Maker and Language Understanding.
- Create a knowledge base.
- Implement multi-turn conversation.
- Test and publish a knowledge base.
- Consume a published knowledge base.
- Implement active learning.
- Create a QnA bot.

To learn more about the QnA Maker service, see the [QnA Maker documentation](/azure/cognitive-services/QnAMaker/).
