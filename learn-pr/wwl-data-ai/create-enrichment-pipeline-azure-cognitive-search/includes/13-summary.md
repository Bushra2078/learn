In this module, you learned how to use Azure Cognitive Search to create an enrichment pipeline that uses AI skills to extract and enhance data for an index. The skills are defined in a *skillset*, and can be:

- *Built-in* skills that leverage Azure Cognitive Services.
- *Custom* skills based on a web service, such as an Azure function.

For more information about Azure Cognitive Search, take a look at the [service documentation](https://docs.microsoft.com/azure/search/).
