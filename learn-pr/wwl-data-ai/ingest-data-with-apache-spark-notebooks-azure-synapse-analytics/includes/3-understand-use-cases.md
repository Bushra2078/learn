It is imperative to understand the use cases for Spark Notebooks within the Synapse Studio environment. 

If you'd like to experiment with your data and gain some insights about the data, notebooks are a good way to start and validate some of the ideas you might have. 

In addition to that, if you want a web interface feel in order to create files that contain live code, you can use the integrated notebook experience in the Synapse studio. 
If you would like to gain some insights in your data through build-in visualizations and text, you are at the right place to start with the notebook experience. 

When you would like to prepare your data and clean it up, it's an integrated experience for creating scripts that can do so. 
You're also free of choice with the languages that you would like to use, which enhances collaboration in your team.
