Microsoft Azure provides a range of technologies for storing non-relational data. Each technology has its own strengths, and is suited to specific scenarios. 

In this module, you've learned about the following technologies, and how you can use them to meet the requirements of various scenarios:

- Azure Table storage
- Azure Blob storage
- Azure File storage
- Azure Cosmos DB
