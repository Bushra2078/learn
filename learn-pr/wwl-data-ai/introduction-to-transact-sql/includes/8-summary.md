In this module, you learned how to:

- Understand what SQL is and how it is used
- Identify database objects in schemas
- Identify SQL statement types
- Use the SELECT statement to query tables in a database
- Work with data types
- Handle NULLs

In this [video series](/shows/programming-databases-with-t-sql-for-beginners/), you can also learn the basics of Transact-SQL. You see how to add, edit, delete, and query data in a relational database. After completing the video series and the accompanying Microsoft Learn modules, you have the foundational knowledge to work with Transact-SQL in your applications.

For more detailed information about Transact-SQL syntax, refer to the [Transact-SQL reference documentation](/sql/t-sql/language-reference).
