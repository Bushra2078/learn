> [!NOTE]
> This exercise requires an Azure subscription. If you do not already have one, sign up for a free trial at [https://azure.microsoft.com](https://azure.microsoft.com?azure-portal=true).

Now, it's your chance to experience how you can use the CLI (v2) to track and manage models with MLflow.

In this exercise, you will:

- Enable autologging to track a model within a command job.
- Use logging functions to track custom metrics for a trained model.

## Instructions

Follow these instructions to complete the exercise:

1. View the exercise repo at [https://aka.ms/aml-cli2](https://microsoftlearning.github.io/mslearn-aml-cli/?azure-portal=true).
1. Complete the **Track Azure ML jobs with MLflow** exercise.
