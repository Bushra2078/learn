As you're preparing to teach this module, get familiar with what the students will learn during the module. In this module, students will learn about Machine Learning including the different forms of Machine Learning. Students will also explore the Azure Machine Learning Service including its features and capabilities. 

This module consists of two lessons:

- **Lesson 1** – Introduction to Machine Learning
- **Lesson 2** – Azure Machine Learning

### Module 2 Tips and Tricks

- Review basic mathematics and statistics like linear regression and logistic regression
   - Assure students this isn't a mathematics course
- Allocate extra time for lab setup
- Use the lab environment to demonstrate the process during the instructional portion of the class
   - Consider assigning other labs as homework assignments or as part of an exam
- Use analogies and examples where possible
- Utilize Microsoft Whiteboard to explain concepts, especially the mathematics
