In this module, you have learned how Azure Synapse Analytics enables you to build data warehouses using modern architecture patterns.

In this module, you have:

- Described a Modern Data Warehouse
- Defined a Modern Data Warehouse Architecture
- Designed ingestion patterns for a Modern Data Warehouse
- Understood data storage for a Modern Data Warehouse
- Prepared and transformed data with Azure Synapse Analytics
- Served data for analysis with Azure Synapse Analytics
