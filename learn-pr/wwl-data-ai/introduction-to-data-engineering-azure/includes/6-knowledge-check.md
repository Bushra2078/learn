## Multiple Choice

Data in a relational database table is…

(x) Structured {{That's correct. Relational databases contain structured data in tables.}}  
() Semi-structured {{That's incorrect. Semi-structured data is typically stored in NoSQL data stores, such as Azure Cosmos DB.}}  
() Unstructured {{That's incorrect. Unstructured data may be stored in a relational database, but a table is not unstructured.}}

## Multiple Choice

In a data lake, data is stored in…

() Relational tables {{That's incorrect. A data lake does not store data in relational tables.}}  
(x) Files {{That's correct. Data in a data lake is stored in files.}}  
() A single JSON document {{That's incorrect. A data lake is based on a distributed file system, not a single JSON document.}}

## Multiple Choice

Which of the following Azure services provides capabilities for running data pipelines AND managing analytical data in a data lake or relational data warehouse?

() Azure Stream Analytics {{That's incorrect. Azure Stream Analytics is used to process real-time streams of data.}}  
(x) Azure Synapse Analytics {{That's correct. Azure Synapse Analytics includes functionality for pipelines, data lakes, and relational data warehouses.}}  
() Azure Databricks {{That's incorrect. Azure Databricks is primarily an Apache Spark implementation for processing data in a data lake.}}
