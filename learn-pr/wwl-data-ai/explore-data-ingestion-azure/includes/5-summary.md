In this module, you've learned about tools for ingesting data into an Azure database. You've seen how to use Azure Data Factory to read, process, and store data in a data warehouse.

In this module, you have learned how to:

* Describe data ingestion in Azure
* Describe components of Azure Data Factory
* Load data into Azure Synapse Analytics
