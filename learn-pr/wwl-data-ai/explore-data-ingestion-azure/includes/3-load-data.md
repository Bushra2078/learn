Imagine that you're part of a team that is analyzing house price data. The dataset that you receive contains house price information for several regions. Your team needs to report on how the house prices in each region have varied over the last few months. To achieve this, you need to ingest the data into Azure Synapse Analytics. You've decided to use Azure Data Factory to perform this task. 

In this video, you'll see how to use Azure Data Factory to ingest and process house price data for analysis. You'll store the data in Azure Synapse Analytics for later analysis.
