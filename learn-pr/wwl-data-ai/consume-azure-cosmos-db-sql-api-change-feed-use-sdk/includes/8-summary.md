In this module, you implemented a basic change feed processor to handle batches of changes to an Azure Cosmos DB for NoSQL container.

Now that you have completed this module, you can:

- Author a change feed processor and delegate in the .NET SDK
- Author a sidecar change feed estimator in the .NET SDK
