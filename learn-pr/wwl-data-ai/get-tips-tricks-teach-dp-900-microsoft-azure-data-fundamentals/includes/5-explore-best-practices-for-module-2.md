

DP-900: Module 2 Explore fundamentals of relational data in Azure

As you're preparing to teach this module, get familiar with what the students will learn during the module. In this module, students will learn about the key characteristics of relational data and explore relational data structures in Azure.
This module consists of two lessons:

- Lesson 1 – Explore relational data concepts
- Lesson 2 – Explore Azure services for relational data
 
> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE5dXUA]

Module 2 Tips and Tricks

- Briefly explain relational data concepts, the characteristics of a relational database, and the pros and cons which will help in the next discussion of non-relational databases
- Describe normalization, views, index, and stored procedures as important relational concepts
- Define the difference between SQL Server on Azure Virtual Machines, Azure SQL Managed Instance, and Azure SQL Database and that it's critical to understand each to make the right choice
- Provide some review questions to test the participants’ understanding
- Practice and prepare the demo before class and if you want your learners to follow along, ask them to set up the demo before the class
