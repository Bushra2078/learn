As you are preparing to teach this module, get familiar with what the students will learn during the module. In this module, students will learn how to use the Text Analytics service to analyze text and how to use the Translator service to translate text. 

This module consists of two lessons: 

- **Lesson 1** – Analyzing text 

- **Lesson 2** – Translating text

### Module 3 Tips and Tricks 

- Explain that NLP allows AI solutions to consume text input, how to analyze it accordingly, and that each API has its own feature and function 
- Discuss that each API has a specific purpose and that there is not one API that can accomplish every task 
  - Consider presenting the labs as demonstrations 
- Explain that you may need to call multiple APIs to accomplish a goal but be wary of calling too many 
- Determine when and when not to call an API.
