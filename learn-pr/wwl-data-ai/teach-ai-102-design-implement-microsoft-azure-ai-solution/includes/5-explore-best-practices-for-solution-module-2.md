As you are preparing to teach this module, get familiar with what the students will learn during the module. In this module, students will learn how to provision, consume, secure, and monitor cognitive services. In addition, students will explore how to deploy and use cognitive services containers. 

This module consists of two lessons: 

- **Lesson 1** – Getting Started with Cognitive Services 

- **Lesson 2** – Using Cognitive Services for Enterprise Applications 

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE4XJof]  

### Module 2 Tips and Tricks 

- Explain how Cognitive Services are structured, development focused, how to consume an API, and how to protect Cognitive Service resources 

- Explain the difference between Multi-service and Single-service 

  - Consider presenting the labs as demonstrations 

- Focus on how to interpret the results, not how to build the model 

- Explain the value to using containers for AI deployments
