As you are preparing to teach this module, get familiar with what the students will learn during the module. In this module, students will learn how to use the Language Understanding (LUIS) service to define a language model and predict intents and entities from user utterances. 

This module consists of three lessons: 

**Lesson 1** – Creating a LUIS app 

**Lesson 2** – Publishing and using a LUIS app 

**Lesson 3** – Using LUIS with speech

### Module 5 Tips and Tricks 

- Discuss topics including natural language processing (NLP) and using Authoring and Prediction resources 
  - Consider presenting the labs as demonstrations 
- Explain the difference between intents vs utterances vs entities 
  - Consider using the example of booking an airline flight to demonstrate 
- Discuss Speech priming
