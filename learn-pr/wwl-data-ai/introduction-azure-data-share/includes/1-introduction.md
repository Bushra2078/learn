Learn how Azure Data Share enables you to send and receive data between departments and external organizations.

After completing this module, you will be able to:
* Describe Azure Data Share
* Provision Azure Data Share
* Share and schedule data using Azure Data Share
* Receive data using Azure Data Share
* Track and monitor data sharing using Azure Data Share
* Secure Azure Data Share
* Manage existing shares

## Prerequisite

Before taking this module, it is recommended that you complete Data Fundamentals.
