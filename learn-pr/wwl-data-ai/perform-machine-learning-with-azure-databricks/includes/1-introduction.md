Imagine that you work as a data scientist for an online retail store. Your organization uses Azure Data Lake to store all its online shopping data and the data engineering team is already using Azure Databricks for day-to-day data-handling operations. Your team has been tasked with exploring machine learning techniques to improve predictive models for customer churn. Your responsibility is to educate the rest of the team on the basics of machine learning, establish a development process the team should adopt as they embark on this new project.

[!INCLUDE [azure-databricks-free-trial-note](../../../includes/azure-databricks-free-trial-note.md)]

## Learning objectives

In this module, you'll:

1. Understand the basics of Machine Learning

1. Understand the Machine Learning Workflow

1. Learn how to use Azure Databricks and PySpark library to implement key parts of the machine learning workflow:

    - Exploratory Data Analysis
    - Modeling
    - Evaluation

## Prerequisites

- An Azure subscription. If you don't have an Azure subscription, create a [free account](https://azure.microsoft.com/free).
