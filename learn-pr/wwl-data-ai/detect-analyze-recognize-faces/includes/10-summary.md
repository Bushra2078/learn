In this module, you have learned how to detect, analyze, and recognize faces.

Now that you've completed this module, you can:

- Identify options for face detection analysis.
- Describe considerations for face analysis.
- Detect faces with the Computer Vision service.
- Describe the capabilities of the Face service.
- Compare and match detected faces.
- Implement facial recognition.

To find out more about face detection with the Computer Vision service, see the [Computer Vision documentation](/azure/cognitive-services/computer-vision/concept-detecting-faces). To find out more about the Face service, see the [Face documentation](/azure/cognitive-services/face/).
