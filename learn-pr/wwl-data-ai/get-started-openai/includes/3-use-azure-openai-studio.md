[Azure OpenAI Studio](https://oai.azure.com/?portal=true) provides access to model management, deployment, experimentation, learning resources, and prompt engineering guidance. You can access the Studio by logging in with your Azure OpenAI resource instance. During the sign-in workflow, select the appropriate directory, Azure subscription, and Azure OpenAI resource.
