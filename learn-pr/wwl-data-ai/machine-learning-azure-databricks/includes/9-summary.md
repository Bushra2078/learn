
Azure Databricks offers a scalable platform with broad support for machine learning workloads. You can use Azure Databricks to cleanse and preprocess data, and to train and evaluate machine learning models by using a wide variety of machine learning frameworks and utilities.

> [!TIP]
> For more information about using Azure Databricks for machine learning workloads, see [Introduction to Databricks Machine Learning](/azure/databricks/machine-learning?azure-portal=true) in the Azure Databricks documentation.
