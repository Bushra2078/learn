Now, it's your chance to explore how to fine-tune a foundation model from the model catalog using the Azure Machine Learning studio.

In this exercise, you will:

- Fine-tune a foundation model for text classification.
- Evaluate the model by reviewing the logged metrics.
- Deploy and test the model in the Azure Machine Learning studio.
