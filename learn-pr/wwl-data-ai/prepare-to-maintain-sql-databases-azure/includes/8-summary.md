
Microsoft provides a great deal of options for deploying SQL Server to Azure. Migrating on-premises databases to Azure SQL Server on Virtual Machine offers portability, while Azure SQL Database and Azure SQL Managed Instance provide scalability and flexibility. Azure Resource Manager (ARM) offers many ways to deploy SQL resources in a both programmatic and graphical fashion. 

Now that you've reviewed this module, you should be able to:

- Understand the role of Azure Database Administrator, and other data platform roles.
- Describe the key differences between the SQL Server-based database options in Azure.
- Describe other open-source SQL database platforms available on Azure.
