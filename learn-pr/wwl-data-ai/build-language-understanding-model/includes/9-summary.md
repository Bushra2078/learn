In this module, you learned how to create a language understanding model.

Now that you've completed this module, you can:

- Provision Azure resources for the Language Understanding service
- Define intents,entities, and utterances
- Use patterns to differentiate similar utterances
- Use pre-built entity components
- Train, test, publish, and review a model

To learn more about Language Understanding, see the [Azure Cognitive Service for Language documentation](/azure/cognitive-services/language-service/?azure-portal=true).
