In this module, you learned how to create a language understanding app.

Now that you've completed this module, you can:

- Provision Azure resources for the Language Understanding service.
- Define intents and utterances.
- Define entities.
- Use patterns to differentiate similar utterances.
- Use pre-built models.
- Train, test, publish, and review a model.

To learn more about Language Understanding, see the [Language Understanding documentation](/azure/cognitive-services/luis/).
