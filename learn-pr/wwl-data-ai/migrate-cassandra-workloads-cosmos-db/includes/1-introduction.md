The Cassandra API for Azure Cosmos DB offers many benefits over an on-premises deployment of Cassandra. This module discusses the benefits of migration and the choices to make, and steps to take, when performing a migration as well as post-migration steps to optimize your system.

## Learning objectives 

By the end of this module, you'll be able to:

- List the benefits of migrating a Cassandra database to Azure Cosmos DB.
- Plan a Cassandra to Cosmos DB migration.
- Explain the steps to perform a migration.
- List the post migration steps to connect your application and optimize your database.
