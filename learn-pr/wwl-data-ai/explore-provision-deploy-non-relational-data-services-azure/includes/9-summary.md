Provisioning is the act of creating an instance of a service. Azure takes care of allocating the resources required to run a service as part of the provisioning process. After you've provisioned a service, you can then configure it to enable your applications and users to access the service. 

In this module, you've learned how to:

- Provision non-relational data services
- Configure non-relational data services
- Explore basic connectivity issues
- Explore data security components
