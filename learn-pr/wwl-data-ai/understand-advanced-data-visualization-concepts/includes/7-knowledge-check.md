## Multiple Choice

(X) Save and share custom theme JSON file.{{Correct. Saving and sharing the JSON file allows others to import and use the theme.}}
( ) Send the modified Power BI (.PBIX) file.{{Incorrect. While this will technically share your theme, it doesn't save it for the new person automatically.}}
() Set the default theme for the organization.{{Incorrect. There is not an organization-wide setting for default themes.}}

## Multiple Choice

What elements can be changed with personalized visuals?
( ) Custom theme color. {{Incorrect. Custom theme color is not modifiable with personalized visuals.}}
(x) Visualization type.{{Correct. Consumers can change visualization type to their preference such as bar to pie chart.}}
( ) Background images.{{Incorrect. Background images are only set through custom themes or at report design.}}

## Multiple Choice

What's an inclusive benefit of designing a report with accessibility features?
( ) Users don't have auto-play videos slowing the performance.{{Incorrect. While videos may impact performance, disabling auto-play allows low vision users to interact with the element at their readiness.}}
( ) Users will admire the bright colors and decorative images used for cohesiveness.{{Incorrect. Cohesive reports are great, but images are not inclusive unless you use alt text as well.}}
(x) Users can understand the report using a screen reader or keyboard.{{Correct. Accessibility features allow high and low vision users to interpret the report equally.}}
