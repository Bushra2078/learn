Imagine that you work as a data scientist for a large online retail store. Your team conducted research and decided on adopting Azure Machine Learning as the platform for machine learning lifecycle management. You have been tasked to understand how Azure Machine Learning supports operationalizing machine learning models. You want to research approaches to deploy and test your models as a scoring web service both in development and in production. You also want to understand how to update models that are already deployed in production.

[!INCLUDE [azure-databricks-free-trial-note](../../../includes/azure-databricks-free-trial-note.md)]

## Learning objectives 

In this module, you'll:

Use mlflow.azureml package and Azure Machine Learning Python SDK to:

* Create or load Azure Machine Learning Workspace
* Register MLflow model with Azure ML and build container image deployment
* Deploy model to Azure Container Instance (ACI)
* Deploy model to Azure Kubernetes Service (AKS)
* Test your model deployments
* Update model deployment in AKS

## Prerequisites

- An Azure subscription. If you don't have an Azure subscription, create a [free account](https://azure.microsoft.com/free).
