Now, it's your chance to experience how you can use the CLI (v2) to create components and a pipeline.   

In this exercise, you will:

- Create components with the CLI (v2). 
- Run a component-based pipeline with the CLI (v2). 
- Create and run a component-based pipeline with the Azure Machine Learning designer.

## Instructions

Follow these instructions to complete the exercise:

1. View the exercise repo at [https://aka.ms/aml-cli2](https://microsoftlearning.github.io/mslearn-aml-cli/?azure-portal=true).
1. Complete the **Run a pipeline with components** exercise.
