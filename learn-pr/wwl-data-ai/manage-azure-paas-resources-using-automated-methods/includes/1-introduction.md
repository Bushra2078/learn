You have learned about some of the capabilities of the SQL Server Agent. However, if you are using Azure SQL Database or Azure Database for MariaDB/MySQL/PostgreSQL, you will have to find alternative scheduling mechanisms, as neither the msdb database nor the SQL Server agent is available. In this module you will learn about Azure automation and elastic jobs, two approaches for automating jobs in PaaS. You will also learn about Azure policy and how it can be used to manage your subscription and costs.

## Learning objectives

At the end of this module, you will understand:

- The benefits of Azure policy
- The capabilities of Azure automation
- How to use Elastic jobs
