You have learned about some of the capabilities of the SQL Server Agent. However, if you are using Azure SQL Database or Azure Database for MariaDB/MySQL/PostgreSQL, you'll need an alternative scheduling mechanism, as both the msdb database and the SQL Server agent are unavailable. In this module you will learn about Azure automation and elastic jobs, two approaches for automating jobs in PaaS. You will also learn about Azure policy and how it can be used to manage your subscription and costs.

## Learning objectives

At the end of this module, you will understand:

- The benefits of Azure policy
- The capabilities of Azure automation
- How to use elastic jobs
