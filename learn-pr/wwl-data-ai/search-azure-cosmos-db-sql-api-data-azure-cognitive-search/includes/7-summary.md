In this module, you explored the Azure Cognitive Search indexer and how it crawls data from Azure Cosmos DB for NoSQL to put in an index. This data was then used for rich search-specific queries that would be prohibitively difficult to implement in a database store.

Now that you have completed this module, you can:

- Index data in Azure Cosmos DB for NoSQL for search queries in Azure Cognitive Search
