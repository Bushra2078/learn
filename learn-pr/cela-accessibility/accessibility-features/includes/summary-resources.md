## Summary

In this module, you have learned how to use accessibility features and tools built into Microsoft products and services. Here are some of the things we covered:

- Windows features and tools, such as keyboard shortcuts, Dictation, Focus assist, and Copilot.
- App features and tools, such as PowerPoint Designer, PowerPoint Presenter Coach, and Microsoft Translator.
