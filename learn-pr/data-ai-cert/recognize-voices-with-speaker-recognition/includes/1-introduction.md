The Speaker Recognition APIs in Azure Cognitive Services provide the most advanced algorithms for speaker verification and speaker identification. They allow you to build voice authorization into applications and identify specific voices.

> [!NOTE]
> This module requires an Azure subscription. The services you create and use are free, but an active subscription or trial will be needed to complete the exercises. If you don't have an Azure subscription, [create a free account](https://azure.microsoft.com/free/) before you begin.

In this module you will:

- Learn what speaker recognition is all about
- Gain an understanding of concepts related to speaker recognition
- Explore the Speaker Recognition API
