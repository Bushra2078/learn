As more data is generated from a variety of connected devices and sensors, transforming this data into actionable insights in near real time is now an operational necessity. Azure Stream Analytics integrates your Internet of Things (IoT) devices and applications with a streaming analytics engine to gain insights with streaming data in real time.

Now that Contoso has gained more confidence with Azure, they're keen to understand how they can integrate real-time analytics into their workflows. This type of analytics is new to the organization.  So the CIO asked you to provide an overview of how "real-time" analytics of streaming data works in principle and how Azure Stream Analytics can provide such capability.

## Learning Objectives

In this module you will:

- Explain data streams
- Explain event processing
- Learn about processing events with Azure Stream Analytics
