Azure Data Lake Storage Gen2 provides a cloud storage service that is highly available, secure, durable, scalable, and redundant. It's the most comprehensive data lake solution available in market today. 

Azure Data Lake Storage brings new efficiencies to process big data analytics workloads and can provide data to a multitude of compute technologies including HDInsight and Azure Databricks without needing to move the data around. Creating an Azure Data Lake Storage Gen2 data store should be one of your go-to tools in building a big data analytics solution.
