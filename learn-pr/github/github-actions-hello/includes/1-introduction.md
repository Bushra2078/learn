To be agile and competitive in today's marketplace, a software development team must automate their software development lifecycle tasks like build, test, and deploy. In this module, you'll learn about how GitHub Actions and GitHub Workflows automate your software development life-cycle tasks for GitHub repositories.

## Learning objectives

In this module, you will:

- Learn about GitHub Actions, the types of actions, and where to find them
- Learn about automating your software development life cycle with GitHub Workflows
- Create a container action and have it run in a workflow triggered by a push event to your GitHub repo

## Prerequisites

- A GitHub account
- The ability to navigate and edit files in GitHub 
  - (For more information about GitHub, see [Introduction to GitHub](https://lab.github.com/githubtraining/introduction-to-github?azure-portal=true).)
