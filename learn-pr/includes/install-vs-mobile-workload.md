> [!NOTE]
> This module requires [Visual Studio 2019](https://visualstudio.microsoft.com/vs/). You can use either Windows or a Mac:
>   - **Windows**: Make sure you have the **Mobile development with .NET** workload installed in Visual Studio 2019 for Windows. Refer to the [documentation](/visualstudio/install/modify-visual-studio?view=vs-2019#modify-workloads) if you need to add it to your installation.
>   - **macOS**: The standard Visual Studio 2019 for Mac installation includes everything you need to build mobile apps with Xamarin.
