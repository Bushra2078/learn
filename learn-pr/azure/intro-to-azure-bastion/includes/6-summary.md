Server administrators understand that they must rely solely on remote management to be able to administer and maintain cloud-based resources, such as Azure VMs. But they want to avoid exposing the remote management interfaces of those VMs to the internet? 

In this module, you learned about how Azure Bastion can help protect your hosted VMs because it avoids the need to expose RDP and SSH ports to the internet. However,Azure Bastion still provides the convenience of securely using both RDP and SSH. You should now be able to determine how you can use Azure Bastion to help secure remote management sessions to your Azure hosted VMs.
