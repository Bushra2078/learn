## Cryptography

![Cryptography category](../media/policies/cryptography.png)

### Why ask these questions

These questions allow you to:

- Learn the cryptographic practices and baselines used to protect the enterprise.
- Know how cryptographic technologies are managed, like Public Key Infrastructure (PKI).
- Understand how Hardware Security Modules (HSM) are deployed, tracked, and administered.

### What questions to ask

|Question|Areas to cover|
|--------|--------------|
|**Describe the systems used to create, manage, and secure cryptographic keys.**|<ul><li>PKI and HSM physical and logical deployment requirements.</li><li>PKI and HSM maintenance.</li><li>Key rotation guidelines.</li></ul>|
|**Describe the systems used to create, manage, and secure certificates.**|<ul><li>Certificate lifecycle.</li><li>Self-signed certificate guidelines.</li></ul>|
|**How is enterprise data protected in-transit, at-rest, and in-use?**|<ul><li>Cryptographic practices and baselines used.</li></ul>|
