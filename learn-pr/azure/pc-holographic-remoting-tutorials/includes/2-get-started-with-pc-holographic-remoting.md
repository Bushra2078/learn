## Spatial awareness

Spatial Awareness provides a set of meshes representing the environment's geometry that allows the engaging interactions between holograms and the actual world. For example, in Mixed Reality applications, the Spatial Awareness system delivers real-world Environmental Awareness.

## Interactable scripts

Objects with interactable scripts respond to different kinds of input interaction states. Also, the script supports various themes that let you alter object parameters like:-

* Color
* Size
* Material
* Shader that affects visual states.
