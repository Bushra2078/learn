# PC Holographic Remoting Tutorials 

 Holographic Remoting allows your applications to send holographic content from a desktop PC or a UWP device to a HoloLens. You can also access additional system resources, allowing you to incorporate remote immersive views into existing desktop PC software.

# Objectives 
 * Learn how to create a Mixed Reality experience demonstration
 *  Learn how to create a PC app for Holographic Remoting

## Prerequisites
* A Windows 10 PC configured with the correct [tools](../../)
* Basic C# programming knowledge
* A HoloLens 2 device [configured for development](../../)
* <a herf="">Unity Hub</a> with Unity 2020/2019 LTS mounted and the Universal Windows Platform Build Support module added.
