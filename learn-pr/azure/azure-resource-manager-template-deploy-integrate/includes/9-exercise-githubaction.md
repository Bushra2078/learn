Here you'll deploy an Azure Resource Manager (ARM) template from a GitHub Action workflow.

> [!IMPORTANT]
> You need your own Azure subscription to run this exercise and you may incur charges. This is required since you will need to create a Service Principle which is **NOT** supported in the Sandbox subscription. If you don't already have an Azure subscription, create a [free account](https://azure.microsoft.com/free/?azure-portal=true) before you begin.
