Ensuring your business online security posture management and protecting it from threats is a growing concern for more businesses by the day. In order to succeed businesses are becoming more and more reliant on multicloud environments to handle the day to day operations of their basic systems and information.

:::image type="content" source="../media/defender-for-cloud-overview.png" alt-text="Screenshot showing a basic overview of Microsoft Defender for Cloud.":::

Supposed you work for Contoso, an online clothing retailer that is developing a new e-commerce website in order to sell its clothes on. Contoso's new clothing website will be made up of several parts, all of which will be hosted in a multicloud environment consisting of Microsoft's Azure, Amazon Web Services (AWS), or Google Cloud Projects (GCP). There will be backend Linux servers that are used to run the websites applications and it will also require SQL databases to store data.

The clothing retailer will want to ensure that its servers and data are all secure. Microsoft Defender for cloud offers various protection plans to ensure the website are secure and protected. For example in this situation, in order to protect the company's website, they would want to enable Microsoft Defender for Cloud to protect them from cyberattacks, using Microsoft Defender for Servers and Microsoft Defender for SQL plans.

:::image type="content" source="../media/benefits.png" alt-text="Screenshot that lists all the different benefits that you will get when you enable Microsoft Defender for Cloud." border="false":::

By the end of this module you will have a better understanding of Microsoft Defender for Cloud and how it can protect your multicloud environments and whether Defender for Cloud is a good choice to protect your multicloud environment from cybersecurity threats.

## Learning objectives

In this module, you will:

- Evaluate whether Defender for Cloud is appropriate to protect your hybrid cloud infrastructure.
- Learn how Defender for Cloud uses Cloud Security Posture Management (CSPM) and Cloud Workload Protection (CWP) to secure your hybrid cloud infrastructure.

## Prerequisites

- Experience with using the Azure at the beginner level.
- Knowledge of cloud security at the beginner level.
