In this module, you learned how to use Event Grid to handle events within your systems. You used an Event Grid trigger within a Logic App to subscribe to events published from Virtual Machine changes, these events enabled you to configure an email notification to be sent. You also discovered how native the implementation of Event Grid is within Azure, and how this allows for simple configuration and usage.
