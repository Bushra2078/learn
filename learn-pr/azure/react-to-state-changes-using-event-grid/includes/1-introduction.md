When you have business-critical applications, you need to a way to manage events and notifications that happen from various systems in an automated way.

Suppose you work on an operations team for a large organization. You have a set of dedicated client virtual machines that are under strict regulatory and change control, and need to be notified if there are any changes to the virtual machines on Azure. You would like to receive emails if there are any changes to these virtual machines, so that you can address any issues as they arise.

You want a way to notify by email any changes performed to your virtual machines.

## Learning objectives

In this module, you will:

- Describe the capabilities and use cases for Event Grid
- Create an application that handles events with Event Grid and Logic Apps
