As you add resources to your Azure Resource Manager templates you will quickly find that you need a way to organize these resources.

Theirs - You tag resources to add values that help you identify their use. For example, you can add tags that list the environment and the project. You could add tags that identify a cost center or the team that owns the resource. Add any values that make sense for your organization.
