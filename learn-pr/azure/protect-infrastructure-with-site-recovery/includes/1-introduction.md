You work for an organization that has recently suffered an outage due to building damage from a hurricane. To prevent this from happening in the future, you've been tasked with moving the infrastructure to Azure as they wish to take advantage of the replication and recovery features it offers. There's a number of virtual machines running in Azure already, and your company now wish to configure protection for them to fail over to a secondary Azure region in the event of a disaster.

In this module, you'll learn about **Azure Site Recovery** and explore it's features for BCDR (Business Continuity and Disaster Recovery) plan.

## Learning objectives

In this module you will:

1. Protect Azure virtual machines with Azure Site Recovery
1. Run a disaster recovery drill to validate protection
1. Failover and failback your virtual machines

## Prerequisites

- Basic understanding of Azure virtual machines
- Basic understanding of Azure virtual networking
