Imagine you're an Azure administrator for a film production company that has 200 TB of footage archived in Azure blob storage. The company plans to re-release the film with some new footage that's in the archive. You've been asked to get a copy of the new material so the producers can work on the film locally. You decide to evaluate Azure Import/Export to export the film footage from Azure blob storage.

In this module, you'll learn about Azure Import/Export and the associated tools that you can use to export your data.

## Learning objectives

In this module, you'll:

- Decide whether to use the Azure Import/Export service.
- Identify the steps required to export data from Azure.

## Prerequisites

- Familiarity with Azure Storage and Azure blobs.
