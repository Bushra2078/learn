# Exercise - Deploy a Microsoft Purview Account

In this unit, you will use the Azure portal to create a Microsoft Purview account.

1. Sign in to the Azure portal, expand the portal menu and click **Create a resource**.  
:::image type="content" source="../media/01.01-create-resource.png" alt-text="Create a resource":::

2. Search the Marketplace for "Microsoft Purview" and click **Create**.  
:::image type="content" source="../media/01.02-create-purview.png" alt-text="Create Microsoft Purview account":::

3. Provide the necessary inputs on the **Basics** tab.
    * Select an existing Azure subscription.
    * Select an existing resource group or create a new resource group.
    * Enter a globally unique Microsoft Purview account name.
    * Select a location.
    * Optionally, overwrite the auto-generated managed resource group name.  
:::image type="content" source="../media/01.03-create-basic.png" alt-text="Microsoft Purview account creation - Basic tab":::

4. On the **Networking** tab, select **All networks**.  
:::image type="content" source="../media/01.04-create-networking.png" alt-text="Microsoft Purview account creation - Networking tab":::

5. On the **Review + Create** tab, once the message in the ribbon returns "Validation passed", verify your selections and click **Create**.  
:::image type="content" source="../media/01.05-create-create.png" alt-text="Microsoft Purview account creation - Networking tab":::

6. Wait several minutes while your deployment is in progress. Once complete, click **Go to resource**.  
:::image type="content" source="../media/01.06-goto-resource.png" alt-text="Microsoft Purview account creation - Networking tab":::
