## Module summary description

Threat Hunting involves proactively seeking out and identifying threat behaviors in the early stages of an attack. In this module you’ll learn how to perform threat hunting using the tools available in Azure Sentinel.

## Subtasks in this module

- How to use queries to hunt threats
- How to use notebooks, bookmarks, and livestream to hunt.

## Module learning objectives

- Explore Sentinel query management and creation
- Hunt threats with notebooks
- Hunt threats with bookmarks
- Hunt threats with livestream
