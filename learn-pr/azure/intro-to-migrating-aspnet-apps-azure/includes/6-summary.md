Hosting ASP.NET applications in private or shared data centers typically result in the organization incurring capital expenditures and ongoing maintenance costs for hardware and software. Migrating the application and its associated data to Microsoft Azure, can provide savings to the organization bu offloading infrastructure costs and automating many aspects of the application maintenance requirements.

The Azure App Service and Azure SQL Database, when combined, provide availability, scale, security, and infrastructure management of your ASP.NET application.

In this module, you have learned what a migration process is and the various tools and options that will help you decide if migration is the right choice. You have also learned how the migration is accomplished in an efficient manner through the use of migration assistants and how integration with a DevOps process can help automate application updates and deployments.
