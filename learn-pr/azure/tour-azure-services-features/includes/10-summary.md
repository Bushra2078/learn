# Summary

You have now learned how to create a free account for Microsoft Azure and log on using that account. You then reviewed the features of the Azure portal and its customization options. Finally, you created, customized, and shared a dashboard.

## Additional Resources

For more information what Azure offers, see [Azure solutions](https://azure.microsoft.com/solutions/)

For a full list of Azure services, see [All Services](https://docs.microsoft.com/azure/#pivot=products&panel=all).

Scroll down the page to see all the services, or click on the sub-sets to the left, such as:

* [Compute](https://docs.microsoft.com/azure/#pivot=products&panel=Compute)
* [Networking](https://docs.microsoft.com/azure/#pivot=products&panel=network)
* [Storage](https://docs.microsoft.com/azure/#pivot=products&panel=storage)

For help on deciding which compute service to use, see [Decision Tree for Azure compute services](https://docs.microsoft.com/azure/architecture/guide/technology-choices/compute-decision-tree).

To associate an Azure subscription to Active Directory, see [How to associate or add an Azure subscription to Azure Active Directory](https://docs.microsoft.com/azure/active-directory/fundamentals/active-directory-how-subscriptions-associated-directory) for how.

For more information on working with Azure Dashboards, see [Programmatically create Azure Dashboards](https://docs.microsoft.com/en-us/azure/azure-portal/azure-portal-dashboards-create-programmatically).
