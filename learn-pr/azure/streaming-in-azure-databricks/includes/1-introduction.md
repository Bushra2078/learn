Suppose you work in the analytics department of a large airlines company. You are part of a team that is working on analyzing the reasons for flight delays based on real-time data continuously being sent by flights around the world. Your job is to analyze the continuous flow of incoming data, that includes details such as Airline, reason for delays, and departure time, and stream it to the data lake storage. You are using Azure Event Hubs to process your data.

## Learning objectives

In this module, you will:

- Use Spark Structured Streaming, Azure Event Hubs, and Databricks Deltaby to read from and write to streams.
- Process streaming data using Azure Databricks.
