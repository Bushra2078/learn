In this module, you learned about Azure Files and Azure File Sync, and how they can help your organization centralize on-premises file shares and provide file storage for cloud-based applications. You also learned about the different types and tiers of Azure file shares, along with their billing model. You learned about the different redundancies available with each type of file share. Then, you wrapped up with a unit on how identity and access control can be used with Azure file shares.
