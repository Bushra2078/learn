Azure Active Directory (Azure AD) is a cloud-based identity and access management solution that helps you secure internal, external, and customer identities.

You work for a global healthcare organization that has around 200 internal staff members. Your organization also has a large volume of health industry customers, and collaborates with healthcare professionals for projects. Your organization has an Active Directory environment on-premises. They're moving to Office 365 and Azure, and you know Azure AD has features that could be beneficial in different use cases. You've been asked to investigate the features available to help secure your organization's identities.

In this module, you'll explore Azure AD and discover why you should use it. You'll look at the various licensing plans and key terminology for Azure AD. Also, you'll investigate the key features and capabilities of Azure AD, and how to deploy it. You'll use the Azure portal to set up and configure Azure AD for your organization.

## Learning objectives

- Describe the core terminology of Azure AD.
- Describe the core features of Azure AD.
- Describe the licensing models for Azure AD.
