In this module, you configured, created, and connected to an Azure Data Science Virtual Machine. You used some of the data-science-oriented software installed on the machine. The Data Science Virtual Machine is available in both Ubuntu and Windows flavors, with much of the same software installed, although some software is available only on one or the other.
