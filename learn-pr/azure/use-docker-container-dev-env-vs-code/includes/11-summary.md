We've covered a variety of important topics in this module to help you create a standard environment for the developers in your agency.

- You have learned how and why you would use a container as a development environment.
- You have installed the Remote - Containers extension in Visual Studio Code and used its commands, such as adding container configuration files, reopening your app in a container, and rebuilding your container after making changes.
- You have explored the files that make up your container configuration.
- You have customized your container and development experience by forwarding ports, changing settings, and installing additional software.
