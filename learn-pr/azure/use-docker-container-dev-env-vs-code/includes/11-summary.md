You've created a standard environment for the developers in your agency that work on the Python Products dashboard. Now, any developer can immediately open, run, and work on this project. You've learned the essential ideas and features of Remote - Containers.

- You've learned how and why you would use a container as a development environment.
- You've installed the Remote - Containers extension in Visual Studio Code and used its commands, such as adding container configuration files, reopening your app in a container, and rebuilding your container after making changes.

- You've customized your container and development experience by forwarding ports, changing settings, and installing additional software.
