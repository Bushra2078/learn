In this module, you learned about Text-to-speech, a cognitive service available on Azure. You learned about the different types of voices that can synthesize text to audio, how the Long Audio API can be used on long bodies of text, and how custom voices can be made and used.

Now that you've reviewed this module, you should be able to:

* Assess the benefits of Text-to-speech
* Describe what Text-to-speech is
* Define scenarios where Text-to-speech can be applied
