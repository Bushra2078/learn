# Module 1 - Modeling Data for Azure Cosmos DB for PostgreSQL

This module identifies considerations that need to be accounted for when modeling tables for Azure Cosmos DB for PostgreSQL. By understanding table classification and sharding and considering your use case, you can create tables for Azure Cosmos DB for PostgreSQL.

## Learning Objectives

In this module, you will:

* Identify key considerations for modeling data for Azure Cosmos DB for PostgreSQL
* Classify tables in Azure Cosmos DB for PostgreSQL
* Create tables within an Azure Cosmos DB for PostgreSQL instance
