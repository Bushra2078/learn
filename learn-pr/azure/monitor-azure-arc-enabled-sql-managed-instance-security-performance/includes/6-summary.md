In this module, you have learned specific methods of Authenticating to an Azure Arc-enabled SQL Managed Instance to meet your security requirements. You have learned what tools are available to assist in the monitoring and optimization of an Azure Arc-enabled SQL Managed Instance.

## Learning objectives

In this module, you have:

- Secured an Azure Arc-enabled SQL Managed Instance
- Configured performance monitoring of an Azure Arc-Enabled SQL Managed Instance
