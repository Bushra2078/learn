Custom Commands is a streamlined design tool for building voice command solutions. Custom Commands is part of Azure Speech Services, a cloud-based resource that provides integrated speech and language understanding technologies.

Custom Commands offers an easy to use application that uses language understanding artificial intelligence, so you can create and customize your own voice assistant application. By using Custom Commands you can add ease to your daily life, as the application is ideal for task completion and command-and-control scenarios, such as for turning your smart home and car devices on and off.

## Prerequisites

* A basic familiarity with Azure

## Learning objectives

In this module, you will:

* Assess the benefits of Custom Commands
* Describe what Custom Commands is
* Define scenarios where Custom Commands can be applied
