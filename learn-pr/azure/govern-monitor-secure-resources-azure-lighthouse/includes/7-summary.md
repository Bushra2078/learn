In this module, you learned how to implement governance, monitor, implement automation, manage hybrid resources, and secure Lamna Healthcare’s resources at Relecloud. You’ve also explored how Lamna Healthcare can view and manage its cloud service providers through Azure Lighthouse.

Without Azure Lighthouse, you couldn't manage Lamna Healthcare’s resources through delegated Service Management. Also, Lamna Healthcare couldn't manage and control access to its resources to managed service providers like Relecloud.
