You want to know how Azure Lighthouse can help a managed services provider like Relecloud and a managed services customer like Lamna Healthcare, to govern, monitor and help secure resources at scale. You'll learn what needs to be done from both the service provider perspective and the customer perspective to implement governance, monitor, implement automation, manage hybrid resources, and secure resources using Azure Lighthouse.

In this module, you'll learn how Azure Lighthouse can help monitor, govern, and secure customer resources.

## Learning objectives

- How to do governance at scale
- How to use centralized monitoring in a scalable way
- How to do security at scale
- How to view and manage service providers (as a customer)
- How to view service provider activity (as a customer)
- How to manage hybrid resources at scale 
- How to automate with Devops with Lighthouse
