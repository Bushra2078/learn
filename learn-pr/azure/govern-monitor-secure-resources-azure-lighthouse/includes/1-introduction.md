You want to know how Azure Lighthouse can help a managed services provider like Relecloud and a managed services customer like Lamna Healthcare, to govern, monitor, and help secure resources at scale. In this module, you'll learn what needs to be done from both the service provider and customer perspectives to implement governance, monitor, implement automation, manage hybrid resources, and secure resources using Azure Lighthouse.

In this module, you'll learn how Azure Lighthouse can help monitor, govern, and secure customer resources.

## Learning objectives

- As a customer, learn how to view & manage service provider activities
- As a managed service provider, learn how to use centralized monitoring in a scalable way
- As a managed service provider, learn how to use governance at scale
- As a managed service provider, learn how to use security at scale
- As a managed service provider, learn how to manage hybrid resources at scale

## Prerequisites

- Have an understanding of Azure Lighthouse
