Visual Studio Team Services (Team Services) provides a highly customizable continuous integration (CI) and continuous deployment (CD) pipeline. When you set up a CI/CD pipeline, each time you make a code commit to source control, Team Services automatically deploys your app to Azure.
  
In this tutorial, you will:
* Learn about the CI/CD process
* Learn about Visual Studio Team Services (VSTS)
* Create a VSTS project


At the end of the tutorial, you can check your knowledge about this information with a quiz challenge.
