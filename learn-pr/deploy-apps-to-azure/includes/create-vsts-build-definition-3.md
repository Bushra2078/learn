To automatically build and deploy your application, you first have to push your code to VSTS:

1. In Visual Studio Team Explorer, select **Changes**:

    ![Commit local changes to the Team Services Git repo](../media/commit-changes.png)

2. In the commit text box, enter a comment like **Initial commit**. In the drop-down menu, select **Commit All and Sync**.

Wasn't that easy? Now, let's create and run a VSTS build. Select **Next**.
