With your web app running in Azure and connected with Visual Studio Team Services (Team Services), now try working with the entire CI/CD pipeline. After you make a change in Visual Studio and commit your code, a build is triggered, which then triggers a release to Azure.

In this tutorial, you will:
- Update an app in Visual Studio and commit changes
- Review the build process and updated a running app

At the end of the tutorial, you can check your knowledge about this information with a quiz challenge.
