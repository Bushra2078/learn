In this module, you have seen how to install SQL Server onto the Ubuntu, SUSE, and Red Hat Linux distributions, and how to deploy SQL Server to a Linux container using Docker. You have deployed SQL Server to Red Hat Enterprise Linux and these steps are very similar for other Linux distributions.

SQL Server on Linux and on Linux containers give you the benefits of the powerful SQL Server database engine without the requirement to change operating system.
