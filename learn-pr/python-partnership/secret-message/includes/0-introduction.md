Unexpectedly, someone walks up to you and hands you a piece of paper that contains a secret, encoded message. The mysterious person tells you that a treasure is hidden at that location, and to meet them there on a specific day to have the treasure revealed. You'll use Python to decode the secret message and find the location and date. 

![Image of an encoded secret message.](../media/encoded-secret-message.png)

Crack the code by using the Python programming language and find the hidden treasure in this beginner's lesson. For the best experience, make sure your local developer environment is ready with [Visual Studio Code](https://code.visualstudio.com/?azure-portal=true). Follow the steps in the [Set up your Python beginner development environment with Visual Studio Code](https://docs.microsoft.com/learn/modules/python-install-vscode/?azure-portal=true) Learn module.

![Screenshot of Python code in Visual Studio Code.](../media/decode-message.png)

## Learning objectives

In this module, you will:

- Write your first lines of Python code
- Create a function to decode a character by using a cipher
- Create a function to decode a word by using a cipher
- Discover the location of the hidden in this module

## Prerequisites

- [Python installed](https://docs.microsoft.com/learn/modules/python-install-vscode/3-exercise-install-python3?azure-portal=true)
- [Visual Studio Code with Python Extension](https://docs.microsoft.com/learn/modules/python-install-vscode/6-exercise-write-execute-first-script?pivots=windows?azure-portal=true)
