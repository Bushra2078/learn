> [!TIP]
> The following keyboard shortcuts are useful within the Azure Cloud Shell editor.
>
> | Keyboard shortcut                                      | Command |
> |--------------------------------------------------------|---------|
> | <kbd>Ctrl+V</kbd> (Windows)<br><kbd>⌘+V</kbd> (macOS) | Paste   |
> | <kbd>Ctrl+S</kbd> (Windows)<br><kbd>⌘+S</kbd> (macOS) | Save    |
