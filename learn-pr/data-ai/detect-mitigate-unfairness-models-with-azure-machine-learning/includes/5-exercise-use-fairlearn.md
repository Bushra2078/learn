Now it's your chance to detect and mitigate unfairness in a model.

In this exercise, you will:

- Use the FairLearn package with Azure Machine Learning to evaluate prediction performance disparity.
- Use the FairLearn package with Azure Machine Learning to mitigate unfairness.

## Instructions

Follow these instructions to complete the exercise.

1. If you do not already have an Azure subscription, sign up for a free trial at [https://azure.microsoft.com](https://https://azure.microsoft.com?azure-portal=true).
2. Open the Lab repo in GitHub at [https://aka.ms/mslearn-aml-labs](https://aka.ms/mslearn-aml-labs?azure-portal=true).
3. If you have not already done so, complete the first two steps in the **Getting Started with Azure Machine Learning** lab to provision an Azure Machine Learning workspace, create a compute instance, and clone the lab files.
4. Complete the **Analyzing and Mitigating Unfairness in Models** lab.
