Imagine that you work as a data scientist for a Bike Share company. Your organization uses Azure Data Lake to store all its historical usage data for New York City. You are responsible for developing a machine learning based regression model to predict seasonal, bike rental demand for New York City.

[!INCLUDE [azure-databricks-free-trial-note](../../../includes/azure-databricks-free-trial-note.md)]

## Learning objectives

In this module, you'll:

1. Understand the three main building blocks in the PySpark's machine learning library: transformers, estimators, and pipelines
1. Learn how build pipelines for common data featurization tasks
1. Deep dive into regression modeling, that include training and interpreting a trained regression model

## Prerequisites

- An Azure subscription. If you don't have an Azure subscription, create a [free account](https://azure.microsoft.com/free).
