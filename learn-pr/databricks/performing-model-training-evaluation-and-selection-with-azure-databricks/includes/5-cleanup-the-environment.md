In this module, you learned how to train your machine learning model using Azure Databricks. To create a high-performing machine learning model, you must train it using different parameters and evaluate its performance against several sample data sets. At the end of this process, you can select the machine learning model that generates the most accurate results and use it for your real-time scenarios.

## Clean up

If you plan on completing other Azure Databricks modules, don't delete your Azure Databricks instance yet. You can use the same environment for the other modules.

## Delete the Azure Databricks instance

1. Navigate to the Azure portal.
1. Navigate to the Resource Group that contains your Azure Databricks instance.
1. Select **Delete resource group**.
1. Type the name of the resource group in the confirmation text box.
1. Select **Delete**.
