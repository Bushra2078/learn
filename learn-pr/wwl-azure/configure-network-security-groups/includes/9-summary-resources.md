Isolating and securing network resources in Azure is an important job skill. You can use network security groups to secure your virtual networks by creating rules to control network traffic.

In this module, you learned how to determine when to use network security groups. You discovered how to evaluate effective network security group rules. You explored how to implement network security group rules
