Azure Digital Twins can be used to build an interactive knowledge graph that represents the full spectrum of interrelationships within your business data.

Suppose you have joined a team that is implementing an Azure IoT solution for the company. The company needs your team to extend the IoT solution to include business data from various sources by implementing Azure Digital Twins. Understanding the components of an Azure Digital Twins solution will help the team to develop an implementation strategy. You decide to examine the architecture and components of an Azure Digital Twins solution.

In this module, you will:

 -  Examine the components of an Azure Digital Twins solution.
 -  Examine Azure Digital Twins models and the Digital Twins Definition Language.
 -  Examine digital twins and graph construction.

By the end of this module, you will be able to describe the requirements for configuring and managing Azure Digital Twins and the primary components within an Azure Digital Twins environment.
