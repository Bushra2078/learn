## Multiple Choice
Users are reporting connectivity errors and timeouts. The help desk thinks a security rule may be blocking traffic to or from one of the virtual machines. To quickly troubleshoot the problem use which Network Watcher feature?
( ) Connection Troubleshoot {{That's incorrect. Connection Troubleshoot troubleshoots network performance and connectivity issues in Azure.}}
(x) IP Flow Verify {{That's correct. IP Flow Verify quickly diagnose connectivity issues from or to the internet and from or to the on-premises environment. For example, confirming if a security rule is blocking ingress or egress traffic to or from a virtual machine. }}
( ) Packet capture {{That's incorrect. Packet capture creates capture sessions to track traffic to and from a virtual machine.}}
