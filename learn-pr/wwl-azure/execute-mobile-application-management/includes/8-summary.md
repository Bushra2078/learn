
This module provided an introduction to Mobile Application Management (MAM), which is a crucial aspect of mobile device management. The module covers key considerations for implementing MAM, including its benefits, challenges, and use cases. It explains the management of MAM in Configuration Manager and how to use Configuration Manager console to manage app protection policies. The module also describes how to use Intune for MAM and implement and manage MAM policies in Intune for different platforms such as iOS, Android, and Windows. By the end of the module, learners will have a comprehensive understanding of MAM and the ability to implement and manage MAM policies in Configuration Manager and Intune. The module's objectives are to explain MAM, understand application considerations in MAM, explain how to use Configuration Manager and Intune for MAM, and implement and manage MAM policies.
