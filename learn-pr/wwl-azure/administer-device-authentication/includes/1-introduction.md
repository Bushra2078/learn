

In this module, you'll learn the importance of device authentication and management when implementing Azure AD in your organization. Like user authentication and identity protection, securing device identities is crucial to maintaining a safe and reliable IT environment. Azure AD provides capabilities for joining Windows-based devices and managing them using distinct methods compared to Active Directory Domain Services. Throughout this module, you'll gain valuable knowledge about device authentication and management within Azure AD.

### Objectives

After completing this module, you'll be able to:

 -  Understand the concept of Azure AD join
 -  Explain the prerequisites, limitations, and benefits of Azure AD join.
 -  Join a device to Azure AD.
 -  Manage devices that are joined to Azure AD.
