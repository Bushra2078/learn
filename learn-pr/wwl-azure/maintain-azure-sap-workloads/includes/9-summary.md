In this module you reviewed how you can connect remotely to Azure VMs running SAP workloads through public endpoints on a Jumpbox VM, or through VPN or Azure ExpressRoute.

You also saw how to back up Azure VMs using the following two methods:

* **Single Azure VM:** Directly from the VM settings in the Azure portal. 
* **Multiple Azure VMs:** Set up a Recovery Services vault and configure backup for multiple Azure VMs.

Lastly, you received an overview for disaster recovery approaches to running SAP workloads in Azure.
