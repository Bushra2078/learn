As you are preparing to teach this module, get familiar with what the students will learn during the module. In this module, students will learn about Identity, Governance, Privacy, and Compliance. Students will learn how Azure Active Directory (AAD) provides identity services and how to govern their subscriptions using services such as Role-based Access Control and policies.  

This module consists of three lessons: 

**Lesson 1** – Secure access to your applications by using Azure identity services 

**Lesson 2** – Build a cloud governance strategy on Azure 

**Lesson 3** – Examine privacy, compliance, and data protection standards on Azure

###Module 5 Tips and Tricks 

- Highlight the importance of identity services 

 - Consider using real life examples and Microsoft [customer stories](https://aka.ms/customerstories) 

- Demonstrate Governance 

 - Consider creating a discussion based on student experiences with governance 

- Demonstrate the compliance websites and discuss their purpose 

 - For example, Microsoft Privacy statement, Trust Center, etc.
