As you are preparing to teach this module, get familiar with what the students will learn during the module. In this module, students will learn about some of the security tools that can help keep infrastructure and data safe when working in the cloud. Students will also explore Azure services that relate to security including Microsoft Defender for Cloud, Microsoft Sentinel, Azure Key Vault, and Azure Dedicated Host. 

This module consists of two lessons: 

- **Lesson 1** – Protect against security threats on Azure. 

- **Lesson 2** – Secure network connectivity on Azure.

### Module 4 Tips and Tricks 

- Consider starting Module 4 with a customer story or short security video related to Azure

 - Consider using the [Microsoft AI demos](https://aka.ms/demosai) 

- Demonstrate Azure security tools and features in the Azure portal 

 - For example, Microsoft Defender for Cloud, Azure Key Vault, etc.  

- Focus on the Defense in Depth model 

 - Break down each layer and explain how services relate to Azure. 

- Incorporate Microsoft Whiteboard or diagrams as needed 

 - Use examples around security solutions such as the Club Azure demonstration from the video
