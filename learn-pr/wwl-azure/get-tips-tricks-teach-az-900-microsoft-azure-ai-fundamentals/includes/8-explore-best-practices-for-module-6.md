As you're preparing to teach this module, get familiar with what the students will learn during the module. In this module, students will learn about the major factors that influence the cost of running in the cloud. Students will explore the cost management tools in Azure and service level agreements (SLA).  

This module consists of two lessons: 

**Lesson 1** – Plan and manage your Azure costs 

**Lesson 2** – Choose the right Azure services by examining SLAs and service lifecycle

###Module 6 Tips and Tricks 

- Demonstrate and compare the [Azure Pricing Calculator](https://aka.ms/pricingcal) and [Total Cost of Ownership (TCO) calculator](https://aka.ms/tcocalc) 

- Explain SLAs 

 - Consider using documentation that defines the SLA for each paid resource in Azure 

- Describe preview services and preview features 

 - Consider using the [Azure updates page](https://aka.ms/updatesaz) to demonstrate the latest updates on Azure products and services that are in development, preview or now available
