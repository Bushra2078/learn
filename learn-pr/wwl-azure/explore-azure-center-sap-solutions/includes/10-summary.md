## Now you can

- Use Azure Center for SAP solutions (ACSS) as an end-to-end solution that enables you to create and run SAP systems as a unified workload on Azure.
- Provide ACSS management capabilities for both new and existing Azure-based SAP systems.
- Deploy single server, distributed, or distributed with High Availability (HA) SAP systems.
- Create an ACSS Virtual Instance for SAP solutions (VIS) resource, which is a logical representation of an SAP system on Azure.
- View the SAP system metadata, get quality checks and insights, monitor and manage your Azure infrastructure, and analyze the cost of running your SAP System on Azure.
