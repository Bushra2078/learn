This module describes Azure App Configuration and details Key-value pairs and App Configuration feature management.

## Learning objectives

After completing this module, students and professionals can:

 -  Describe Azure App Configuration.
 -  Understand Key-value pairs.
 -  Understand app configuration feature management.
 -  Implement application configuration.

## Prerequisites

 -  Understanding of what DevOps is and its concepts.
 -  Familiarity with version control principles is helpful but isn't necessary.
 -  Beneficial to have experience in an organization that delivers software.
