## Scenario

Your company needs you to ensure there are systems in place to quickly detect and troubleshoot issues for the SAP HANA on Azure (Large Instances) workloads that are ready to go live. You work with your administrators and the support team to build in robust monitoring and alerts, and set up the diagnostic tools needed to troubleshoot HANA Large Instances issues efficiently.

## Learning objectives

You will:

 -  Explore SAP HANA on Azure (Large Instances) monitoring.
 -  Explore troubleshooting and monitoring SAP HANA on Azure (Large Instances).
 -  Explore SAP HANA alerts.
 -  Explore diagnostic tools.
 -  Learn how to raise support requests for SAP HANA on Azure (Large Instances).
