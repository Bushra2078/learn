
Configuring and maintaining your network security is critical for a safe and secure Azure solution. Use tools like Network Security Groups, Front Door, and others to control and lock down access to your data.

You should be able to:

 -  Deploy and configure network security groups to protect your Azure solutions
 -  Configure and lockdown service endpoints and private links
 -  Secure your applications with Application Gateway, Web App Firewall, and Front Door
 -  Configure ExpressRoute to help protect your network traffic
 -  Explain how network security acts as one layer in a defense in depth strategy
