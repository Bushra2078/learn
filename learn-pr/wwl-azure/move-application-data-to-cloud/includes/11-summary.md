In this module, we investigated how to migrate data to an Azure cloud solution. We looked at tools that you can use to help make the transition and then options on Azure to ensure your migrated data is safe and performant.

With the information in this module, your cloud migration may be able to better optimize your overall data solution making it easier to manage for you and perform better for your users.

In this module, you learned how to:

* Describe types of data and identify which Azure data solution may be the best fit for your needs
* Identify some of the security features Azure's data solutions offer
* Optimize SQL query performance
* Migrate data to Azure
