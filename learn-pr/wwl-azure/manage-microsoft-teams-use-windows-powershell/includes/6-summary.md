In this module you've learned how to use the Microsoft Teams PowerShell module to perform tasks such as creating teams and managing user permissions. The following are the key takeaways:

- The Microsoft Teams module helps manage Microsoft Teams by using Windows PowerShell. It includes many functions for configuring of the overall service, but not individual Teams. 
- The Microsoft Teams module can be installed from the PowerShell Gallery. When you connect to Microsoft Teams, be sure to sign in with a user account that has sufficient privileges to perform the actions you want to complete.
- The Microsoft Teams PowerShell module can be used to create, configure, and manage teams, user settings, and channels.
