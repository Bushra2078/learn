Azure Resource Health can help you monitor whether your Azure Digital Twins instance is up and running. You can also use it to learn whether a regional outage is impacting the health of your instance.

:::image type="content" source="../media/m11-l04-adt-resource-health-portal-143a0427.png" alt-text="Screenshot that shows how to configure Azure Resource Health to help you monitor whether your Azure Digital Twins instance is up and running.":::


In the image above, the Azure Digital Twins instance is showing the current state as Available, and you can see that the resource has been Available for the previous nine days.

To learn more about the Available status and the other status types that may appear, see [Azure resource health overview](/azure/service-health/resource-health-overview).
