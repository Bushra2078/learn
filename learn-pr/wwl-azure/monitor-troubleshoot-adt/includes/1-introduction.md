An Azure Digital Twins solution can be monitored using Azure Monitor and metrics provided by the Azure Digital Twins service.

Suppose you have joined a team that is implementing an Azure IoT solution for the company. The company needs your team to extend the IoT solution to include business data from various sources by implementing Azure Digital Twins. Understanding how to monitor and troubleshoot the Azure Digital Twins will help to ensure that the solution is supported appropriately. You decide to examine the Azure Digital Twins metrics, logs, and alerts that are used to support the solution.

In this module, you will:

 -  Examine the Azure Digital Twins metrics.
 -  Examine the Azure Digital Twins diagnostics settings, logs, and alerts.
 -  Review Azure Resource Health support for Azure Digital Twins.

By the end of this module, you will be able to describe the tools and processes that can be used to monitor an Azure Digital Twins solution.
