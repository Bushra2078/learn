Azure Time Series Insights can be used to visualize, analyze, and store time series data, and it integrates well with Azure IoT solutions.

Suppose you have joined a team that is implementing an Azure IoT solution for the company. Your team will be supporting downstream business integrations and efforts to generate insights from IoT data. Understanding the capabilities and requirements of Azure Time Series Insights will help the team to ensure successful implementation of the service. You decide to examine the Azure Time Series Insights resource configuration and integration requirements.<br>

In this module, you will:

 -  Examine the use cases and benefits of Azure Time Series Insights.
 -  Examine Time Series Insights resource configuration.
 -  Examine Time Series Insights integration with IoT Hub.

By the end of this module, you will be able to describe the requirements for integrating Azure Time Series Insights within an IoT solution.
