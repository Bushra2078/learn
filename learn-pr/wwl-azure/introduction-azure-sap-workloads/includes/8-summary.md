In this module, you learned the three options for deploying SAP products in Azure are Azure VMs, SAP HANA on Azure (Large Instances), and the SAP Cloud Appliance Library (CAL). You also examined the SAP HANA certified Azure storage types used in SAP HANA deployments and reviewed the network considerations for deploying SAP workloads to Azure. Lastly, you saw how Azure facilitates authentication and authorization including the support of a variety of identity providers.
