The Azure portal provides access to Azure resources and enables you to customize the user interface so that you are more productive.

Suppose you have joined a team that will be implementing an Azure IoT solution for the company. The team is new to Azure and the Azure portal. You decide to spend some time getting used to the Azure portal before the project kicks off.

In this module, you will:

 -  Review features of the Azure portal.
 -  Explore the Azure portal toolbar, navigation menu, and dashboard.

By the end of this module, you will be able to customize the Azure portal for your projects.
