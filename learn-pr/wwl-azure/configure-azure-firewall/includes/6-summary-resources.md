Azure Firewall acts as a barrier between your Azure virtual network implementation and the internet. Azure Firewall examines all inbound and outbound traffic. Azure Firewall uses threat intelligence, rules, and other policy settings to allow legitimate traffic and deny threatening or unknown traffic.

In this module, you reviewed characteristics and features of Azure Firewall. You learned how to evaluate when to use the service to secure your Azure virtual networks. You explored how to implement Azure Firewall and how to configure firewall rules.
