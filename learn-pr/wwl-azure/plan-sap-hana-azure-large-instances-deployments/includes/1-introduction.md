## Scenario

Your company needs to address the unique needs of SAP HANA on Azure (Large Instances). You guide your SAP on HANA Large Instances deployment team through the details of various components and scenarios that need to be considered. The team starts the planning for ExpressRoute and supporting multiple SAP systems.

## Learning objectives

You will:

 -  Explore SAP HANA on Azure (Large Instances) compute considerations.
 -  Examine SAP HANA on Azure (Large Instances) storage considerations.
 -  Examine SAP HANA on Azure (Large Instances) networking considerations.
 -  Explore SAP HANA on Azure (Large Instances) ExpressRoute FastPath.
 -  Examine SAP HANA on Azure (Large Instances) single and multiple SAP systems.
 -  Explore SAP HANA on Azure (Large Instances) routing in Azure.
 -  Explore SAP HANA on Azure (Large Instances) supported scenarios.
