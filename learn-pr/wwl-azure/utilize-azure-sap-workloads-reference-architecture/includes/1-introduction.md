This module provides the reference architectures and proven practices for running SAP NetWeaver in a Windows environment on Azure with high availability, running S/4HANA in a high availability environment that supports disaster recovery on Azure, and running SAP HANA on Azure (Large Instances) with high availability and disaster recovery (DR).



## Learning objectives 

In this module you will:

* Understand the reference architecture and proven practices for running SAP NetWeaver in a Windows environment on Azure with high availability

* Explore the reference architecture for running S/4HANA in a high availability environment that supports disaster recovery on Azure

* Learn how to utilize SAP HANA workloads on Azure Large Instances

* Walk-through a sample workflow of SAP HANA workloads on Azure Large Instances
