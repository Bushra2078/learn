Azure Defender for IoT security agents and custom alerts can be used to enhance the security of your IoT solution by monitoring events on IoT devices.

Suppose you have joined a team that is implementing an Azure IoT solution for the company. Your team needs to enhance the solution to meet the company's security requirements. You decide to gain experience by implementing Azure Defender for IoT and custom security alerts in a test environment.

In this module, you will complete a lab that includes the following activities:

 -  Enable Azure Defender for IoT and create a security module twin.
 -  Install a security agent on a device.
 -  Create and trigger a custom alert.
 -  Review the alert in Azure Defender for IoT.

By the end of this module, you will be able to implement Azure Defender for IoT security agents and custom alerts on a Linux device.
