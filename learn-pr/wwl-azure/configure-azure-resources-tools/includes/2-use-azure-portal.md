The **Azure portal** lets you build, manage, and monitor everything from simple web apps to complex cloud applications in a single, unified console.

:::image type="content" source="../media/azure-portal-0949f400.png" alt-text="Screenshot of the Azure portal showing the cloud shell icon.":::


 -  Search resources, services, and docs.
 -  Manage resources.
 -  Create customized dashboards and favorites.
 -  Access the Cloud Shell.
 -  Receive notifications.
 -  Links to the Azure documentation.

> [!NOTE]
> You can access the portal at [https://portal.azure.com](https://portal.azure.com/).
