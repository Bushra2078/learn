As you are preparing to teach this module, get familiar with what the students will learn during the module. In this module, students will begin to understand the different tools and techniques that they can use to administer Azure. Students will learn about the Azure Resource Manager (ARM) and explore Azure Administrator tools including Azure portal, Azure Cloud Shell, Azure PowerShell, and Azure Command Line Interface (CLI). Additionally, students will understand the operations of ARM templates and how they relate to the resources in Microsoft Azure.   

This module consists of three lessons: 

- **Lesson 1** – Configure Azure Resources with Tools 

- **Lesson 2** – Use ARM 

- **Lesson 3** – Configure ARM Templates

### Module 3 Tips and Tricks 

- Discuss where Azure originated  

- Explain that there are multiple ways to accomplish tasks in Azure 

- Discuss that different skill level and tool knowledge can be applied in Azure 

- Emphasize the importance of being comfortable with ARM templates 

  - Consider using demonstrations. For example, students create resources and look at the templates used to create the resource
