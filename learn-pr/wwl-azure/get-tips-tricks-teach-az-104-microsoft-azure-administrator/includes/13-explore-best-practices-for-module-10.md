As you're preparing to teach this module, get familiar with what the students will learn during the module. In this module, students will explore data protection. Topics covered include file and folder backups and Virtual Machine (VM) protections.  

This module consists of two lessons: 

- **Lesson 1** – Configure file and folder backups 

- **Lesson 2** – Configure VM backups 

> [!VIDEO https://www.microsoft.com/videoplayer/embed/RE4XWEo]

### Module 10 Tips and Tricks 

- Emphasize that Azure offers backup services and replication services for data protection 

- Discuss that Azure has built in services for data protection but also supports third party services 

- Explain that data sovereignty is an idea that the data will always be owned and controlled by the customer 

- Consider presenting the labs as demonstrations
