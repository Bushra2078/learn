As you're preparing to teach this module, get familiar with what the students will learn during the module. In this module, students will learn about governance and compliance. Students will explore obtaining and managing Azure subscriptions. Additionally, students will learn about Azure Resource Manager and apply their knowledge about how it's used to manage Azure Policy and Azure Role-Based Access Control (RBAC).   

This module consists of three lessons: 

**Lesson 1** – Configure Subscriptions 

**Lesson 2** – Configure Azure Policy 

**Lesson 3** – Configure RBAC

### Module 2 Tips and Tricks 

- Consider starting the conversation on governance and compliance by discussing Azure regions and the diverse legal and regulatory compliance requirements implemented around the world 

- Explain that policies control resources and RBAC control user access to avoid confusion 

  - Consider completing the demonstrations of both Azure Policy and RBAC 

- Emphasize the difference between RBAC roles and Azure AD roles 

   - Consider presenting the labs as demonstrations
