As you are preparing to teach this module, get familiar with what the students will learn during the module. In this module, students will learn about network routing and endpoints. Additionally, students will explore Azure Load Balancer and Azure Application Gateway.  

This module consists of three lessons: 

- **Lesson 1** – Configure Network Routing and Endpoints 

- **Lesson 2** – Configure Azure Load Balancer 

- **Lesson 3** – Configure Azure Application Gateway

### Module 6 Tips and Tricks 

- Understand that Module 6 builds on Module 4 and Module 5 

  - Complete modules in sequential order 

- Discuss that network traffic comes from a variety of different sources 

 - Emphasize managing the traffic to the networks and balancing the traffic to virtual machines or applications 

- Understand that connectivity and security are the underlying drivers for everything in networking 

- Labs require additional provisioning before they can be completed	 

  - Allow additional time for lab setup
