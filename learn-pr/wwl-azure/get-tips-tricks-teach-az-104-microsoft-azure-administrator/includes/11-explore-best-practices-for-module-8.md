As you are preparing to teach this module, get familiar with what the students will learn during the module. In this module, students will learn about Azure virtual machines (VM). Topics covered will include Azure VM availability and Azure VM extensions.   

This module consists of three lessons: 

- **Lesson 1** – Configure Azure VM 

- **Lesson 2** – Configure Azure VM availability 

- **Lesson 3** – Configure Azure VM extensions

### Module 8 Tips and Tricks 

- Module 8 is the second longest module in the class 

- Discuss that VMs provide the base for Infrastructure as a Service (IaaS) in Azure. 

- Explain IaaS and Platform as a Service (PaaS) 

- This module includes 3 hours of labs 

  - Allow plenty of time to cover the content and labs 

- Utilize Microsoft Whiteboard during instruction 

- Consider presenting the labs as demonstrations
