As you are preparing to teach this module, get familiar with what the students will learn during the module. In this module, students will learn about monitoring. Topics covered will include Azure Monitor, Azure Alerts, Log Analytics, and Network Watcher tools. 

This module consists of four lessons: 

- **Lesson 1** – Configure Azure Monitor 

- **Lesson 2** – Configure Azure Alerts 

- **Lesson 3** – Configure Log Analytics 

- **Lesson 4** – Configure Network Watcher

### Module 11 Tips and Tricks 

- Emphasize that Azure Monitor is in every other module in the course 

  - Discuss that Azure Monitor is the “glue” that ties the principles of this Azure course together 

  - Consider revisiting other Modules to find examples of when Azure Monitor is doing work for that resource 

- Understand Azure Monitor offers a unified monitoring platform for Azure, on-premises, and other clouds. Diverse data sources can be connected using Azure Monitor agent and logged data collected in this one platform 

- Discuss that Azure Monitor collects the data for Azure Advisor recommendations 

- The data collected by Azure Monitor is used by a number of different services 

- Consider presenting the labs as demonstrations
