As you are preparing to teach this module, get familiar with what the students will learn during the module. In this module, students will learn about Intersite Connectivity. Students will identify use cases and product features of virtual network peering as well as configure gateway transit, connectivity, and service chaining. Students will also explore how to create and configure Azure Virtual Private Network (VPN) Gateway and new concepts like Azure ExpressRoute.  

This module consists of three lessons: 

- **Lesson 1** – Configure Virtual Networks (VNet) Peering 

- **Lesson 2** – Configure VPN Gateway 

- **Lesson 3** – ExpressRoute and Virtual Wide Area Network (WAN)

### Module 5 Tips and Tricks 

- Understand that Module 4 builds into Module 5 

  - Build out the resources in Module 4 and connect them in Module 5 

- Labs in this module require additional provisioning before they can be completed 

  - Allow time for lab provisioning 

  - Begin with demonstration of how to set up the lab environment 

- Emphasize that the key difference between VNet Peering and VPN as the encryption 

- Consider presenting the labs as demonstrations
