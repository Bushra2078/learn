As you are preparing to teach this module, get familiar with what the students will learn during the module. In this module, students will explore Azure Storage Accounts. Topics covered will include blob storage, options for storage security, Azure Files and File Sync, and Azure Storage management tools.  

This module consists of five lessons: 

- **Lesson 1** – Configure Azure Storage Accounts 

- **Lesson 2** – Configure blob storage 

- **Lesson 3** – Configure Azure Storage security 

- **Lesson 4** – Configure Azure Files and File Sync 

- **Lesson 5** – Configure Azure Storage with tools

### Module 7 Tips and Tricks 

- Emphasize that Azure Storage is one of three top-level categories of services in Azure: Compute, Storage, and Network 

- Module 7 is one of the longest modules in the course 

  - This module can be broken up into multiple sections 

- Understand that the material is straightforward, but the concepts of storage, tiering, replication, and security are evolving and may be confusing for those who are new to the cloud 

- Consider using demonstrations  

  - For example, have students create storage accounts
