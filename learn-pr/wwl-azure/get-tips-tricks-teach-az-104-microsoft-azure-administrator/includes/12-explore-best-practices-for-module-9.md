As you are preparing to teach this module, get familiar with what the students will learn during the module. In this module, students will explore serverless computing. Topics covered will include Azure App Service plans, Azure App Services, Azure Container Instances, and Azure Kubernetes Service (AKS).   
 
This module consists of four lessons: 

**Lesson 1** – Configure Azure App Service plans 

**Lesson 2** – Configure Azure App Services 

**Lesson 3** – Configure Azure Container Instances 

**Lesson 4** – Configure AKS

### Module 9 Tips and Tricks 

- Discuss different options for pricing and consumption in Azure 

- Discuss the benefits of App Services vs Containers 

- Emphasize that Azure is not creating containers, it is supporting existing Docker containers on a Docker platform running in Azure 

- Explain that AKS is not Google’s Kubernetes, Azure has built their own Kubernetes services with full featured orchestration for containers 

- Focus on Kubernetes services use of Docker’s platform to run containers 

- Allow plenty of time to complete labs 

   - Spend time with students to ensure they understand concepts from the labs 

   - After completion of the labs, consider allowing time for discussion
