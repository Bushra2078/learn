# Introduction

In this module, you’ll be introduced to cloud service types. You’ll learn how each cloud service type determines the flexibility you’ll have with managing and configuring resources. You'll understand how the shared responsibility model applies to each cloud service type, and about various use cases for each cloud service. You’ll also learn about some typical use cases for each of the cloud service models.

# Learning objectives

After completing this module, you’ll be able to:

 -  Describe infrastructure as a service (IaaS).
 -  Describe platform as a service (PaaS).
 -  Describe software as a service (SaaS).
 -  Identify appropriate use cases for each cloud service (IaaS, PaaS, SaaS).
