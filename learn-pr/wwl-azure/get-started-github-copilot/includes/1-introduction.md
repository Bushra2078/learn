This module includes an overview of AI-assisted programming tools from Microsoft and GitHub, followed by an in-depth examination of GitHub Copilot. You examine GitHub Copilot benefits, account types, features, and limitations. After completing a step-by-step installation of the GitHub Copilot extensions for Visual Studio Code, you explore the features presented by the GitHub Copilot and GitHub Copilot Chat tools. You also complete an exercise that provides guidance on configuring the GitHub Copilot extensions in Visual Studio Code.

Imagine you're a software developer at a tech startup. You're tasked with developing a feature for one of the company's newest apps. You're happy to get the assignment, but the deadline is tight, you haven't seen the codebase, and you need to ensure the new feature is bug-free and efficient. You heard that GitHub Copilot can accelerate your development process, so you're looking forward to using it. GitHub Copilot can explain new and complex code, generate documentation, and help you develop code more efficiently and accurately. Using GitHub Copilot will help you meet your deadline and deliver a high-quality feature to your team.

The topics covered in this module include:

- Examining AI tools from GitHub and Microsoft that assist application developers.
- Understanding GitHub Copilot: tools, features, account types, and limitations.
- Installing and using GitHub Copilot extensions in Visual Studio Code.
- Exploring the autocomplete features of GitHub Copilot in Visual Studio Code.
- Exploring the AI-interaction features of GitHub Copilot Chat in Visual Studio Code.
- Configuring GitHub Copilot extensions in Visual Studio Code.

By the end of this module, you're able to use GitHub Copilot to enhance your productivity, reduce errors, and make your software development process more efficient.

> [!IMPORTANT]
> To complete this GitHub Copilot training, you must have an active subscription for GitHub Copilot in your personal GitHub account, or you need to be assigned to a subscription managed by an organization or enterprise.
