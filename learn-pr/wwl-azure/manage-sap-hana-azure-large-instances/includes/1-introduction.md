## Scenario

Your company is completing deployments of several SAP HANA on Azure (Large Instances) workloads. You must finalize maintenance tasks and procedures so the various administration teams in your company will be prepared for maintaining HANA Large Instances.

## Learning objectives

You will:

 -  Connect to Azure HANA on Azure (Large Instances) from Azure virtual machines and from on-premises locations.
 -  Manage Azure HANA on Azure (Large Instances) via the Azure portal.
 -  Configure networking changes for SAP HANA on Azure (Large Instances).
 -  Configure operating system and workload updates for SAP HANA on Azure (Large Instances).
 -  Set up SMT-based updates for SUSE Linux.
 -  Scale SAP HANA on Azure (Large Instances).
