This module describes how to implement routing for mobile applications and configure App Center Diagnostics and alerts.

## Learning objectives

After completing this module, students and professionals can:

 -  Implement routing for mobile applications.
 -  Configure App Center Diagnostics.
 -  Configure alerts.
 -  Create a bug tracker.

## Prerequisites

 -  Understanding of what DevOps is and its concepts.
 -  Familiarity with version control principles is helpful but isn't necessary.
 -  Beneficial to have experience in an organization that delivers software.
