This module introduced you to GitHub Packages. It explored ways to control permissions and visibility, publish, install, delete and restore packages using GitHub.

You learned how to describe the benefits and usage of:

 -  Publish packages.
 -  Install packages.
 -  Delete and restore packages.
 -  Configure access control and visibility.
