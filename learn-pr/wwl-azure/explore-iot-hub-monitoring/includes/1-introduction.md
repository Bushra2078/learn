The Azure IoT Hub service provides metrics that can be monitored using Azure monitor to help ensure that your solution is performing as expected.

Suppose you have joined a team that is implementing an Azure IoT solution for the company. Your team will need to configure IoT Hub monitoring and troubleshoot device connection/communication issues that may occur. You decide to gain experience by configuring IoT Hub monitoring in a test environment.

In this module, you will complete a lab that includes the following activities:

 -  Enable IoT Hub diagnostics logs and metrics.
 -  Configure alerts for IoT Hub metrics.
 -  Trigger alerts and verify records in the diagnostics log.

By the end of this module, you will be able to configure IoT Hub monitoring and log alerts.
