This module introduces you to Azure Kubernetes Service (AKS) and its configurations, architectural components, Ingress controllers, network, and deployment units. You will learn how to implement continuous deployment for AKS.

## Learning objectives

After completing this module, students and professionals can:

 -  Implement Azure Kubernetes Service (AKS).
 -  Deploy and connect to an AKS cluster.
 -  Implement continuous deployment with AKS.
 -  Update container images.

## Prerequisites

 -  Understanding of what DevOps is and its concepts.
 -  Familiarity with version control principles is helpful but isn't necessary.
 -  Beneficial to have experience in an organization that delivers software.
