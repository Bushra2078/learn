In this module, you walked through the planning and implementation considerations for SAP installations in Azure. We reviewed the network considerations for deploying of SAP workloads on Azure, and reviewed the RDBMS platforms that are supported by SAP for running workloads in Azure. Lastly, we examined four methods for provisioning storage for SAP workloads in Azure:
* Azure VM disks
* Azure VM NFS storage
* Azure VM SMB storage
* SAP HANA on Azure (Large Instance) storage
