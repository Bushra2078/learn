SAP offers customer-specific sizing for SAP HANA with TDIv5 (Tailored Datacenter Integration version 5). The list of SAP HANA on Azure (Large Instances) is available for deployment with TDIv5 authorization from SAP. With these instances, you get more memory per core, allowing for a cost-effective implementation. You must work with SAP to get certification for their specific project and workload.

TDIv5 is available in:

 -  US West and US East
 -  Australia East
 -  Australia Southeast
 -  Japan East
 -  Japan West
 -  West Europe
 -  North Europe
