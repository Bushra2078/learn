SAP-certified operating systems supported for SAP products on SAP HANA on Azure (Large Instances) include:

## HANA Large Instance Type I

- SLES 12 SP3, SP4, SP5
- SLES 15 SP1
- RHEL 7.6

### Persistent memory SKUs

- SLES 12 SP4

## HANA Large Instance Type II

- SLES 12 SP3, SP4, SP5
- SLES 15 SP1
- RHEL 7.6

> [!NOTE]
> For more information, see [Compatible operating systems for HANA Large Instances](/azure/virtual-machines/workloads/sap/os-compatibility-matrix-hana-large-instance).
