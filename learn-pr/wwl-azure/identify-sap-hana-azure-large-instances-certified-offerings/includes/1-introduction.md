## Scenario

You need to prepare for the kickoff meeting for planning for migration one of your largest SAP HANA workloads to Azure that will require **SAP HANA on Azure (Large Instances)** (also known as **HANA Large Instances** or **HLI**). SAP HANA on Azure (Large Instances) is a unique solution on Azure offering the possibility to run and deploy SAP HANA on dedicated bare-metal servers. You will introduce your team to the foundations of HANA Large Instances solutions and discuss the supported configurations.

## Learning objectives

You will:

- Explore SAP HANA on Azure (Large Instances).
- Explore deployment options for SAP solutions on Azure.
- Review available SKUs for SAP HANA on Azure (Large Instances).
- Explore custom sizing for SAP HANA on Azure (Large Instances).
- Review SAP-certified configurations.
- Examine SAP HANA certifications.
- Review SAP HANA on Azure (Large Instances) supported operating systems.
- Explore SAP HANA on Azure (Large Instances) storage.
