In this exercise, we will introduce using *Azure Data Migration Services* (***DMS***) to migrate to an Azure Cosmos DB API for MongoDB account. While we will use a MongoDB Backup for this migration, similar steps will be taken to migrate from a live MongoDB database.

> [!NOTE]
> To complete this exercise, you'll need a Microsoft Azure subscription. If you don't already have one, you can sign up for a free trial at ***https://azure.microsoft.com/free***. You cannot use a Microsoft Learn sandbox subscription for this exercise..
