Microsoft's Power BI product is a powerful data analytics tool that can connect to a wealth of data sources to visualize data and share insights.

Suppose you have joined a team that is implementing an Azure IoT solution for the company. Your team will be supporting downstream business integrations and efforts to generate insights from IoT data. Understanding how data visualizations are used by analysts to generate insights will help the team to configure IoT data appropriately for input to the visualization service. You decide to examine the capabilities of Power BI and its connection requirements.

In this module, you will:

 -  Review Power BI options and capabilities.
 -  Examine Power BI data connections.
 -  Examine Power BI data visualization options.

By the end of this module, you will be able to describe the analytics and data visualization capabilities introduced by Power BI.
