Azure Database for MySQL has many features to keep your data secure. In this module, you'll learn about the Azure Database for MySQL security features including role-based access control (RBAC), encryption, and how to configure Azure Active Directory authentication. 

After completing this module, you'll be able to:

- Describe authentication
- Understand super user access
- Describe Azure RBAC for Azure Database for MySQL
- Create users in Azure Database for MySQL
- Understand encryption in Azure Database for MySQL
- Understand Azure Active Directory authentication
