Azure Database for MySQL has comprehensive security features to keep your server and data safe. These include encryption, authentication, and Role Based Security. You can also use Azure AD authentication with Azure Database for MySQL single server.

Now that you have completed this module, you can:

- Describe authentication
- Understand super user access
- Describe Azure RBAC for Azure Database for MySQL
- Create users in Azure Database for MySQL
- Understand encryption in Azure Database for MySQL
- Understand Azure Active Directory authentication.
