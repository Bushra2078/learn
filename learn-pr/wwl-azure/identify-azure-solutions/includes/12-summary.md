## Summary

In this module, you explored the many different Azure solutions, avaiable to help build your applications.  From Internet of Things (IoT) to Artificial Intelligence and many others, Azure provides the tools you need.  During this module you built out an IoT Hub and saw a Raspberry Pi device communicate with it, implemented an Azure Function, and built your first web app in the cloud.

Each of these solution help you build compelling applications for your business needs and for your customers.
