## Multiple Choice
Which of the following Project boards types can contain issues and pull requests from any personal repository?
(x) User-owned project boards. {{Correct. User-owned project boards can contain issues and pull requests from any personal repository.}}
( ) Organization-wide project boards. {{Incorrect. Organization-wide project boards can contain issues and pull requests from any repository that belongs to an organization.}}
