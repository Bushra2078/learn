This module provided you with the necessary knowledge to harden Windows Server. You learned how to ensure that domain joined computers in your environment had unique local administrator passwords, how to secure administrative access with Privileged Access Workstations, how to secure Active Directory Domain Services Domain Controllers, how to analyze security configuration with Security Compliance Toolkit and how to secure SMB traffic.

Now that you have reviewed this module, you should be able to:

- Manage local administrator passwords using Local Administrator Password Solution

- Limit administrative access to Privileged Access Workstations (PAWs)

- Explain how to secure domain controllers from being compromised

- Describe how to use the Microsoft Security Compliance Toolkit to harden servers

- Secure SMB traffic using SMB encryption
