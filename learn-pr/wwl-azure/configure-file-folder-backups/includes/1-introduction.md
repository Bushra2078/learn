The Azure Backup service provides a simple, secure, and cost-effective solution for backing up your data. Administrators implement the Microsoft Azure Recovery Services (MARS) agent for Azure Backup to recover their data from the Microsoft Azure cloud.

You work for a large enterprise in the finance industry. Your company stores critical compliance information on Azure Files file shares. You need a reliable and cost-effective backup solution that can protect your data and ensure quick recovery. You're responsible for ensuring this content can be recovered if there's data loss or corruption. You need to configure backup and restore policies that meet your company's regulatory needs.

In this module, you learn about the features and usage cases of Azure Backup. You learn how to create and configure options for an Azure Recovery Services vault. You review the process for on-premises file and folder backups. 

The goal of this module is to provide you with the knowledge and skills to effectively use Azure Backup. Azure Backup protects your data in the cloud and on-premises.

## Learning objectives

In this module, you learn how to:

- Identify features and usage cases for Azure Backup.

- Configure Azure Recovery Services vault backup options.

- Implement on-premises and cloud file and folder backup and recovery.

- Configure the Microsoft Azure Recovery Services agent for Azure Backup.

## Skills measured

The content in the module helps you prepare for [Exam AZ-104: Microsoft Azure Administrator](/certifications/exams/az-104). 

## Prerequisites

- Familiarity with general backup concepts, including types of backups, backup schedules, and recovery options.

- Able navigate and use the Azure portal. This skill is essential for backing up resources and services.
