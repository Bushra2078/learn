Azure Files file share backup is a native, cloud-based backup solution that protects your data in the cloud. Azure Backup eliminates extra maintenance overhead involved in on-premises backup solutions. The Azure Backup service smoothly integrates with Azure File Sync. You can centralize your file share data along with your backups.

In this module, you identified features and usage cases for Azure Backup. You reviewed how to create and configure options for an Azure Recovery Services vault. You explored how to implement on-premises file and folder backups. You discovered how to configure the Microsoft Azure Recovery Services (MARS) agent.

The main takeaways from this module are:

- Azure Backup simplifies the process of backing up your on-premises and cloud resources.

- The Backup center for Azure Backup lets enterprises govern, monitor, operate, and analyze their backups in Azure. 

- Azure Backup integrates with existing Azure services to enable management at scale and provides detailed reports on backups. 

- Azure Backup uses the MARS agent to back up files and folders from on-premises machines and Azure VMs. 

- Backups are stored in Recovery Services vault.
