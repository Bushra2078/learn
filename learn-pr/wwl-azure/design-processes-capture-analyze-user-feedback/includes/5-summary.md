This module introduced you to the Twitter sentiment release gate and detailed deployment with release management.

You learned how to describe the benefits and usage of:

 -  Create a release gate for the Twitter sentiment hashtag.
 -  Design processes to capture user feedback.
 -  Deploy with release management.
 -  Create alerts using other release gates.
