## Best Practices related to Creating Classes and Properties

Supplement or modify the following best practices for your own work situations:

- Use properties instead of accessing fields in a class directly.
- Make sure each class has only one responsibility. Don't try to create a class that does it all.
- Organize your classes into namespaces.
