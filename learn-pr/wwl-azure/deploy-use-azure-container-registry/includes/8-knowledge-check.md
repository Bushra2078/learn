## Multiple Choice
What is the purpose of Azure Container Registry?
( ) To create and manage virtual machines {{Incorrect. Azure Container Registry is not used to create and manage virtual machines.}}
( ) To store and manage data in a cloud database {{Incorrect. Azure Container Registry is not used to store and manage data in a cloud database.}}
(x) Enables the automation and validation of the creation and teardown of environments to help deliver secure and stable application hosting platforms {{Correct. Azure Container Registry is a private registry service for building, storing, and managing container images and related artifacts. It enables the automation and validation of the creation and teardown of environments to help deliver secure and stable application hosting platforms.}}
