In this exercise, you'll  build a knowledge mining solution that makes it easy to search for insights about customer experiences. You'll create an Azure Cognitive Search index using data extracted from customer reviews.

> [!NOTE]
> To complete this lab, you will need an [Azure subscription](https://azure.microsoft.com/free?azure-portal=true) in which you have administrative access.

Launch the exercise and follow the instructions.
